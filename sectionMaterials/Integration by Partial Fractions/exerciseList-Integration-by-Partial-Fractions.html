\(\ds \frac{2}{(x + 3)(x - 2)}\)
The rational expression contains two linear factors: \((x + 3)\) and \((x - 2).\) The partial fraction decomposition then takes the shape \[\frac{2}{(x + 3)(x - 2)} = \frac{A}{x + 3} + \frac{B}{x - 2} \period\] Manipulating these fractions, we have \[ \ba \frac{2}{(x + 3)(x - 2)} &= \frac{A}{x + 3} \left(\frac{x - 2}{x - 2}\right) + \frac{B}{x - 2} \left(\frac{x + 3}{x + 3}\right) \\[1.5ex] 2 &= A(x - 2) + B(x + 3) \period \ea \] Our objective is now to solve for \(A\) and \(B.\) Letting \(x = 2\) yields \[2 = 0 + 5B \implies B = \frac{2}{5} \period\] Similarly, for \(x = -3\) we get \[2 = -5A + 0 \implies A = -\frac{2}{5} \period\] We have, therefore, \[\frac{2}{(x + 3)(x - 2)} = \boxed{\frac{-2/5}{x + 3} + \frac{2/5}{x - 2} }\]
\(\ds \frac{3x - 1}{x^2 - 8x - 9}\)
We factor the denominator as \[\frac{3x - 1}{(x - 9)(x + 1)} \comma\] from which our partial fraction decomposition takes the form \[ \frac{3x - 1}{(x - 9)(x + 1)} = \frac{A}{x - 9} + \frac{B}{x + 1} \, \] Algebraic manipulation gives \[ \ba \frac{3x - 1}{(x - 9)(x + 1)} &= \frac{A}{x - 9} \left(\frac{x + 1}{x + 1}\right) + \frac{B}{x + 1} \left(\frac{x - 9}{x - 9}\right) \\[1.5ex] 3x - 1 &= A(x + 1) + B(x - 9) \period \ea \] We now perform substitutions to solve for \(A\) and \(B.\) Letting \(x = -1\) gives \[-4 = 0 - 10B \implies B = \frac{2}{5} \period\] Also, assigning \(x = 9\) produces \[26 = 10A + 0 \implies A = \frac{13}{5} \period\] We now have, for our final answer, \[\frac{3x - 1}{(x - 9)(x + 1)} = \boxed{\frac{13/5}{x - 9} + \frac{2/5}{x + 1}} \]
\(\ds \frac{2x + 3}{9x^2 - x} \)
Factoring the denominator, we have \[\frac{2x + 3}{x(9x - 1)} \comma\] so our decomposition shape is \[\frac{2x + 3}{x(9x - 1)} = \frac{A}{x} + \frac{B}{9x - 1} \period\] Adding these partial fractions results in \[2x + 3 = A(9x - 1) + Bx \period\] We now let \(x = 0\) to get \[3 = -A + 0 \implies A = -3 \period\] Furthermore, letting \(x = 1/9\) gives \[\frac{2}{9} + 3 = 0 + \frac{1}{9} B \implies B = 29 \period\] Thus, we obtain \[\frac{2x + 3}{9x^2 - x} = \boxed{\frac{-3}{x} + \frac{29}{9x - 1} }\]
\(\ds \frac{t^2}{8t^3 - 1}\)
The denominator is a difference of cubes that can be factored to obtain \[\frac{t^2}{8t^3 - 1} = \frac{t^2}{(2t - 1)(4t^2 + 2t + 1)} \period\] The quadratic \((4t^2 + 2t + 1)\) is nonreducible, so our decomposition is in the form \[\frac{t^2}{(2t - 1)(4t^2 + 2t + 1)} = \frac{A}{2t - 1} + \frac{Bt + C}{4t^2 + 2t + 1} \period\] Adding the partial fractions gives \[t^2 = A(4t^2 + 2t + 1) + Bt(2t - 1) + C(2t - 1) \period\] We now solve for \(A,\) \(B,\) and \(C.\) Two of the terms contain the factor \((2t - 1),\) so letting \(t = 1/2\) zeroes them. We attain \[\frac{1}{4} = 3A + 0 \implies A = \frac{1}{12} \period\] We now have two unknowns: \[t^2 = \frac{1}{12} (4t^2 + 2t + 1) + Bt(2t - 1) + C(2t - 1) \period\] Letting \(t = 0\) provides \[0 = \frac{1}{12} + 0 - C \implies C = \frac{1}{12} \period\] At this point, our remaining unknown is \(B;\) we have \[t^2 = \frac{1}{12}(4t^2 + 2t + 1) + Bt(2t - 1) + \frac{1}{12}(2t - 1) \period\] We assign \(t\) any value to directly solve for \(B.\) The simplest value is \(t = 1,\) which gives \[1 = \frac{7}{12} + B + \frac{1}{12} \implies B = \frac{1}{3} \period\] Our final answer is \[ \frac{t^2}{8t^3 - 1} = \boxed{\frac{\frac{1}{12}}{2t - 1} + \frac{\frac{1}{3}t + \frac{1}{12}t}{4t^2 + 2t + 1} }\]
\(\ds \frac{x^3 + 1}{9 - x^2}\)
We cannot perform partial fraction decomposition immediately, since the degree of the numerator is greater than the degree of the denominator. So our first step is to perform polynomial long division. Doing so gives the simplified expression \[-x + \frac{9x + 1}{9 - x^2} \period\] Factoring the denominator of \(\frac{9x + 1}{9 - x^2}\) gives \[\frac{9x + 1}{(3 + x)(3 - x)} \period\] Our partial fraction decomposition is then in the form \[\frac{9x + 1}{(3 + x)(3 - x)} = \frac{A}{3 + x} + \frac{B}{3 - x} \period\] Adding the partial fractions, we obtain \[9x + 1 = A(3 - x) + B(3 + x) \period\] Now we let \(x = 3 \! :\) \[28 = 0 + 6B \implies B = \frac{14}{3} \period\] Then letting \(x = -3\) gives \[-26 = 6A + 0 \implies A = -\frac{13}{3} \period\] Therefore, our final answer is \[\frac{x^3 + 1}{9 - x^2} = \boxed{-x - \frac{13/3}{3 + x} + \frac{14/3}{3 - x} }\]
\(\ds \frac{3 - 10x^3}{5x^3 + 8x^2 - 11x} \)
We begin by performing polynomial division because the degrees of the numerator and denominator are equal. Doing so gives \[\frac{3 - 10x^3}{5x^3 + 8x^2 - 11x} = -2 + \frac{16x^2 - 22x + 3}{5x^3 + 8x^2 - 11x} \period \] Our objective is now to find the partial fraction decomposition of \(\frac{16x^2 - 22x + 3}{5x^3 + 8x^2 - 11x} \!:\) \[ \ba \frac{16x^2 - 22x + 3}{5x^3 + 8x^2 - 11x} &= \frac{16x^2 - 22x + 3}{x(5x^2 + 8x - 11)} \\ &= \frac{A}{x} + \frac{Bx + C}{5x^2 + 8x - 11} \period \ea \] We then have \[16x^2 - 22x + 3 = A(5x^2 + 8x - 11) + Bx^2 + Cx \period\] To solve for \(A,\) we let \(x = 0\) to get \[3 = -11A + 0 + 0 \implies A = -\frac{3}{11} \period \] There is no convenient substitution to solve for \(B\) or \(C.\) Thus, we should arrange a system of equations to solve for these unknowns. Letting \(x = 1,\) for simplicity, gives \[ -3 = -\frac{6}{11} + B + C \implies B + C = -\frac{27}{11} \period \] Also, letting \(x = -1\) provides the simple equation \[41 = -14 \left(-\frac{3}{11}\right) + B - C \implies B - C = \frac{409}{11} \period\] Our system is then \[ \ba B + C &= -\frac{27}{11} \\[1.5ex] B - C &= \frac{409}{11} \comma \ea \] for which the solutions are \[B = \frac{191}{11} \qquad \textrm{and} \qquad C = -\frac{218}{11} \period \] Our final answer is then \[ \ba \frac{3 - 10x^3}{5x^3 + 8x^2 - 11x} &= \boxed{\frac{-3/11}{x} + \frac{191x - 218}{11(5x^2 + 8x^2 - 11)} } \ea \]

\(\ds \int \frac{4 - 3x}{(7x + 3)(4 - x)} \, \dd x\)
The partial fraction decomposition of the integrand takes the shape \[\frac{4 - 3x}{(7x + 3)(4 - x)} = \frac{A}{7x + 3} + \frac{B}{4 - x} \period\] We then get \[4 - 3x = A(4 - x) + B(7x + 3) \period\] We solve for \(B\) by letting \(x = 4 \!:\) \[-8 = 0 + 31B \implies B = -\frac{8}{31} \period\] Letting \(x = -3/7\) yields \[4 + \frac{9}{7} = A \left(4 + \frac{3}{7} \right) \implies A = \frac{37}{31} \period\] We have, therefore, \[ \ba \int \frac{4 - 3x}{(7x + 3)(4 - x)} \, \dd x &= \int \frac{37/31}{7x + 3} - \frac{8/31}{4 - x} \, \dd x \\[1.5ex] &= \boxed{\frac{37}{217} \ln|7x + 3| + \frac{8}{31} \ln|4 - x| + C } \ea \]
