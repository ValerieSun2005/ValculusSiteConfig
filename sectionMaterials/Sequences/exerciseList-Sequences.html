What is the difference between a sequence and a series? Provide an example of each.
A sequence is a list of numbers—for example, \[\{1, 2, 3, \dots\} \pd\] A series is a sum of numbers—for example, \[1 + 2 + 3 + \cdots \pd\]

\(\{-2, 2, -2, 2, \dots\}\)
The terms of the sequence alternate and never settle to any final value. Thus, the sequence diverges.
\(\ds \left\{\frac{n - 2}{n + 5} \right\} \subsuper{1}{\infty}\)
We take the limit as \(n \to \infty\) of the \(n\)th term: \[\lim_{n \to \infty} \frac{n - 2}{n + 5} = 1 \pd\] Therefore, the sequence converges to \(1.\)
\(\ds \{0, -3, -6, -9, \dots\}\)
As \(n \to \infty,\) the terms of the sequence approach \(-\infty.\) Thus, the sequence diverges.
\(\ds \{b_n\} \subsuper{0}{\infty} \cma b_n = \frac{n^2 + 4n - 8}{3n^4 + 5}\)
We find \[\lim_{n \to \infty} b_n = \lim_{n \to \infty} \frac{n^2 + 4n - 8}{3n^4 + 5} = 0 \pd\] Thus, sequence \(\{b_n\}\) converges to \(0.\)
\(\ds \{a_n\} \subsuper{1}{\infty} \cma a_n = \frac{n^3 - 8n^2 + 12n - 5}{9n^2 + 5n - 12}\)
We find \[\lim_{n \to \infty} \frac{n^3 - 8n^2 + 12n - 5}{9n^2 + 5n - 12} = \infty \pd\] Since \(\{a_n\}\) does not approach any value, the sequence diverges.
\(\ds \left\{\frac{3}{4}, \frac{9}{16}, \frac{27}{64}, \frac{81}{256}, \dots\right\}\)
The general term of this sequence is \[a_n = \par{\frac{3}{4}}^n \cma\] starting from \(n = 1.\) Then \(\lim_{n \to \infty} a_n = 0,\) so the sequence converges to \(0.\)

determine whether the Divergence Test establishes divergence for the series

\(\ds \sum_{n = 2}^\infty \frac{2n^2}{5n^2 + 11n - 7}\)
We see \[\lim_{n \to \infty} \frac{2n^2}{5n^2 + 11n - 7} = \frac{2}{5} \pd\] This limit doesn't equal \(0,\) so the Divergence Test states that \(\sum_{n = 2}^\infty \frac{2n^2}{5n^2 + 11n - 7}\) diverges.
\(\ds \sum_{n = 0}^\infty \cos n\)
The function \(\cos n\) oscillates continuously as \(n \to \infty,\) never settling to any final value. The limit \(\lim_{n \to \infty} \cos n\) therefore does not exist. So the Divergence Test asserts that the sum diverges.
\(\ds \sum_{n = 1}^\infty e^n \cdot 3^{-n}\)
We rewrite the series as \[\sum_{n = 1}^\infty \par{\frac{e}{3}}^n \pd\] Then \[\lim_{n \to \infty} \par{\frac{e}{3}}^n = \lim_{n \to \infty} \frac{e^n}{3^n} = 0 \pd\] [The denominator's base in \((e/3)^n\) is greater than the numerator's, so the denominator grows faster and thus forces the limit to \(0.\)] Because this limit is \(0,\) the Divergence Test is inconclusive.
\(\ds 2 + \frac{1}{2} + \frac{1}{8} + \frac{1}{32} + \cdots\)
The sum can be expressed in summation notation as \[\sum_{n = 0}^\infty 2 \par{\frac{1}{4}}^n \pd\] We see \[\lim_{n \to \infty} 2 \par{\frac{1}{4}}^n = \lim_{n \to \infty} \frac{2}{4^n} = 0 \pd\] Thus, the Divergence Test is inconclusive.
\(\ds \sum_{n = 1}^\infty \frac{n!}{n^{1234567890}}\)
It is a fact that factorials grow faster than do exponential functions. Thus, \[\lim_{n \to \infty} \frac{n!}{n^{1234567890}} = \infty \pd\] Since this limit doesn't equal \(0,\) the sum diverges.
If \(\lim_{n \to \infty} a_n = 0,\) then \(\sum a_n\) converges.
This statement is false. The Divergence Test asserts that \(\sum a_n\) diverges if \(\lim_{n \to \infty} a_n \ne 0;\) the test stands silent when \(\lim_{n \to \infty} = 0.\)
If \(\{a_n\}\) diverges, then \(\sum a_n\) also diverges.
This statement is true. If \(\{a_n\}\) diverges, then \(\lim_{n \to \infty} a_n\) does not exist. The Divergence Test states that \(\sum a_n\) diverges if \(\lim_{n \to \infty} a_n \ne 0.\)
If \(\{a_n\}\) converges, then \(\sum a_n\) also converges.
This statement is false. If \(\{a_n\}\) converges, then \(\lim_{n \to \infty} a_n\) equals a finite value. But this says nothing about \(\sum a_n \col\) If \(\lim_{n \to \infty} a_n \ne 0,\) then the Divergence Test states that \(\sum a_n\) diverges. And if \(\lim_{n \to \infty} a_n,\) then the test is inconclusive. For example, the sequence \[\{1, 1, 1, \dots\}\] converges to \(1.\) But the sum \[1 + 1 + 1 + \cdots\] diverges.
Consider the family of sums \[S = \sum_{n = 1}^\infty a_n + \sum_{n = 1}^\infty b_n \pd\]
  1. Find formulas for \(a_n\) and \(b_n\) such that \(\sum_{n = 1}^\infty a_n\) and \(\sum_{n = 1}^\infty b_n\) both diverge and \(S\) diverges.
  2. Find formulas for \(a_n\) and \(b_n\) such that \(\sum_{n = 1}^\infty a_n\) and \(\sum_{n = 1}^\infty b_n\) both diverge and \(S\) converges.
  1. If \(a_n = n\) and \(b_n = n,\) then both \[\sum_{n = 1}^\infty a_n \and \sum_{n = 1}^\infty b_n\] diverge by the Divergence Test because both \(\lim_{n \to \infty} a_n\) and \(\lim_{n \to \infty} b_n\) approach \(\infty.\) Then \[S = \sum_{n = 1}^\infty n + \sum_{n = 1}^\infty n \pd\] The terms of \(S\) are \[\par{1 + 2 + 3 + \cdots} + \par{1 + 2 + 3 + \cdots} \cma\] a clearly divergent series. Thus, \(a_n = n\) and \(b_n = -n\) represent one satisfactory option.
  2. If \(a_n = n\) and \(b_n = -n,\) then both \[\sum_{n = 1}^\infty a_n \and \sum_{n = 1}^\infty b_n\] diverge by the Divergence Test because both \(\lim_{n \to \infty} a_n\) and \(\lim_{n \to \infty} b_n\) are unbounded and therefore do not equal \(0.\) These choices of \(a_n\) and \(b_n\) show \[ \ba S &= \sum_{n = 1}^\infty n + \sum_{n = 1}^\infty (-n) \nl &= \sum_{n = 1}^\infty n - \sum_{n = 1}^\infty n \nl &= 0 \pd \ea \] So \(S\) is convergent. Thus, \(a_n = n\) and \(b_n = -n\) represent one satisfactory option.
