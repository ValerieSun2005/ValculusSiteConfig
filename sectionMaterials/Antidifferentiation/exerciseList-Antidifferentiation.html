Let \(F\) be an antiderivative of function \(f.\) What property does \(F\) satisfy? How many functions \(F\) exist?
By definition of an antiderivative, \(F' = f.\) Moreover, \(f\) has infinitely many antiderivatives \(F\)—all of which differ by a constant.
In \(\int f(x) \di x = F(x) + C,\) define each symbol.
  • \(\int\) is called the integral sign;
  • \(f(x)\) is called the integrand;
  • \(\dd x\) is called the differential;
  • \(F(x)\) is an antiderivative of \(f;\) and
  • \(C\) is called a constant of integration.

evaluate the integral

\(\ds \int \sin x \di x\)
The derivative of \(-\cos x\) is \(\sin x.\) Thus, \(-\cos x\) is an antiderivative of \(\sin x.\) The general antiderivative of \(\sin x\) is \(-\cos x + C,\) where \(C\) is a constant. So \[\int \sin x \di x = \boxed{- \cos x + C}\]
\(\ds \int \frac{1}{\sqrt{1 - x^2}} \di x\)
Since the derivative of \(\sin^{-1} x\) is \(1/\sqrt{1 - x^2},\) an antiderivative of \(1/\sqrt{1 - x^2}\) is \(\sin^{-1} x.\) The general antiderivative of \(1/\sqrt{1 - x^2}\) is \(\sin^{-1} x + C;\) therefore, \[\int \frac{1}{\sqrt{1 - x^2}} \di x = \boxed{\sin^{-1} x + C}\]
\(\ds \int \sec \theta \tan \theta \di \theta\)
An antiderivative of \(\sec \theta \tan \theta\) is \(\sec \theta.\) We then add a constant of integration in our final answer: \[\int \sec \theta \tan \theta \di \theta = \boxed{\sec \theta + C}\]
\(\ds \int x^6 \di x\)
Using the reverse power rule, \[\int x^6 \di x = \frac{x^{6 + 1}}{6 + 1} + C = \boxed{\tfrac{1}{7} x^7 + C}\]
\(\ds \int \sqrt[3]{x} \di x\)
We write the integrand as \(x^{1/3}\) and antidifferentiate using the reverse power rule: \[\int x^{1/3} \di x = \frac{x^{1 + 1/3}}{1 + 1/3} + C = \boxed{\tfrac{3}{4} x^{4/3} + C}\]
\(\ds \int \frac{1}{x^3} \di x\)
Writing the integrand as \(x^{-3},\) we antidifferentiate using the reverse power rule: \[\int x^{-3} \di x = \frac{x^{-3 + 1}}{-3 + 1} + C= -\tfrac{1}{2} x^{-2} + C = \boxed{-\frac{1}{2x^2} + C}\]
\(\ds \int 3y - 5 \di y\)
By the linearity property of integrals, we split the integral as \[\int 3y \di y- \int 5 \di y \pd\] The same property allows us to pull out the constants in each integral, so we obtain \[3 \int y \di y - 5 \int \di y \pd\] Using the reverse power rule then gives \[\boxed{\tfrac{3}{2} y^2 - 5y + C}\]
\(\ds \int 9 \di t\)
We pull out the constant \(9 \col\) \[9 \int \di t = \boxed{9t + C}\]
\(\ds \int 2t - \cos t \di t\)
Since integrals exhibit the property of linearity, we split the integral. Breaking the integral into two and pulling out constants give \begin{equation} 2 \int t \di t - \int \cos t \di t \pd \label{eq:2t-cost-dt} \end{equation} We use the reverse power rule to evaluate the first integral, and the second integral (without regard to the negative sign) is \(\sin t.\) Thus, \(\eqref{eq:2t-cost-dt}\) becomes \[2 \frac{t^{1 + 1}}{1 + 1} - (\sin t) + C = \boxed{t^2 - \sin t + C}\]
\(\ds \int \pi^2 \di x \)
The integrand, \(\pi^2,\) is a constant. Thus, we pull out the constant to obtain \[\pi^2 \int \di x = \boxed{\pi^2 x + C}\]
\(\ds \int u \sqrt u \di u\)
We write the integrand as \(u \cdot u^{1/2} = u^{3/2}.\) Then using the reverse power rule, we obtain, as the general antiderivative, \[\frac{u^{1 + 3/2}}{1 + 3/2} + C = \boxed{\tfrac{2}{5} u^{5/2} + C}\]
\(\ds \int \frac{3}{x} + \frac{1}{\sqrt{x}} - 3x^2 \di x\)
We express all terms as powers. We then break up the integral into three and pull out constants: \begin{equation} 3 \int x^{-1} \di x + \int x^{-1/2} \di x - 3 \int x^2 \di x \pd \label{eq:3/x+} \end{equation} The first integral is \(\ln|x|,\) while the others require the reverse power rule. Thus, \(\eqref{eq:3/x+}\) becomes \[3 \ln|x| + \frac{x^{1 - 1/2}}{1 - 1/2} - 3 \frac{x^{2 + 1}}{2 + 1} + C = \boxed{3 \ln|x| + 2 \sqrt x - x^3 + C}\]
\(\ds \int (2x - 3)(x + 4) \di x\)
We expand the integrand as \[2x^2 + 5x - 12 \pd\] We then integrate this expression using the reverse power rule, obtaining \[\boxed{\tfrac{2}{3} x^3 + \tfrac{5}{2} x^2 - 12x + C}\]
\(\ds \int \frac{3x^3 + 6x^2 - \sqrt x}{x^2} \di x\)
We write the integral as \[\int 3x + 6 - \frac{\sqrt x}{x^2} \di x = \int 3x + 6 - x^{-3/2} \di x \pd\] Using the reverse power rule on each term, we obtain \[\boxed{\tfrac{3}{2} x^2 + 6x + \frac{2}{\sqrt x} + C}\]

\(\ds g'(t) = 3t^2 - 4 \cos t \cma g(0) = 2\)
We integrate \(g',\) getting \begin{equation} g(t) = t^3 - 4 \sin t + C \pd \label{eq:diff-eq-3t^2} \end{equation} To solve for \(C\) we substitute in \(\eqref{eq:diff-eq-3t^2}\) the initial condition \(\orange{g(0) = 2} \col\) \[\orange{g(0)} = (0)^3 - 4 \sin 0 + C = \orange{2} \implies C = 2 \pd\] We plug \(C = 2\) into \(\eqref{eq:diff-eq-3t^2}\) to obtain the particular solution: \[g(t) = \boxed{t^3 - 4 \sin t + 2}\]
\(\ds \deriv{p}{x} = e^x + 2x \cma p(1) = 2\)
Antidifferentiating \(p'\) gives \begin{equation} p(x) = e^x + x^2 + C \pd \label{eq:diff-eq-p(x)} \end{equation} In \(\eqref{eq:diff-eq-p(x)}\) we plug in the initial condition, \(\orange{p(1) = 2},\) to get \[\orange{p(1)} = e^1 + (1)^2 + C = \orange{2} \implies C = 1 - e \pd\] Thus, substituting this value of \(C\) in \(\eqrefer{eq:diff-eq-p(x)}\) confers the particular solution, \[p(x) = \boxed{e^x + x^2 + 1 - e}\]

A penny is dropped from the roof of a \(50\)-foot building. Find \(h(t),\) the penny's height above the ground as a function of time, in seconds. Determine the penny's speed right before it hits the ground.
Since the penny is dropped, its initial velocity is \(v(0) = 0.\) Because we want the penny's height function, we let the upward direction be positive; thus, the object's acceleration due to gravity is \(-g = -32 \undiv{ft}{sec}^2.\) We integrate this value to obtain the penny's general velocity function: \[v(t) = \int -32 \di t = -32t + C \pd\] To solve for \(C\) we substitute the initial condition \(\orange{v(0) = 0} \col\) \[\orange{v(0)} = -32(0) + C = \orange 0 \implies C = 0 \pd\] Thus, the penny's velocity function is \[v(t) = -32t \pd \] The penny's height function is \(h(t) = \int v(t) \di t,\) and this model takes the initial condition \(\orange{h(0) = 50}.\) Integrating, we have \[h(t) = -16t^2 + D \pd\] To solve for \(D\) we substitute the initial condition \(\orange{h(0) = 50}:\) \[\orange{h(0)} = -16(0)^2 + D = \orange{50} \implies D = 50 \pd\] Hence, the penny's height at time \(t\) is \[h(t) = \boxed{-16t^2 + 50}\] To find the penny's final velocity \(v_f,\) we find the time at which \(h(t) = 0.\) We get \[-16t^2 + 50 = 0 \implies t = \frac{5}{2 \sqrt 2} \pd\] Then \[v_f = v \par{\frac{5}{2 \sqrt 2}} = -32 \par{\frac{5}{2 \sqrt 2}} = -40 \sqrt 2 \pd\] Speed is the magnitude (absolute value) of velocity, so the penny's final speed is \(|-40 \sqrt 2|\) \(\approx 56.569 \undiv{ft}{sec}.\)
A pencil rolls off a desk that is \(6\) feet above the ground. How long does the pencil take to hit the ground?
We need to choose an axis orientation. It is logical to let up be positive; then the pencil's acceleration due to gravity is \(-32 \undiv{ft}{sec}^2.\) We find the pencil's velocity function by integrating this value of acceleration with respect to time: \[v(t) = \int -32 \di t = -32t + C \pd\] The pencil rolls off the desk, so its initial velocity is \(\orange{v(0) = 0}.\) Substituting this initial condition gives \[\orange{v(0)} = -32(0) + C = \orange 0 \implies C = 0 \pd\] Hence, the pencil's velocity function is \(v(t) = -32t.\) The pencil's height function is the integral of \(v(t)\) with respect to time—that is, \[h(t) = \int -32 t \di t = -16t^2 + D \pd\] The pencil initially is \(6\) feet above the ground; thus, \(\orange{h(0) = 6}.\) Substituting this initial value shows \[\orange{h(0)} = -16(0)^2 + D = \orange 6 \implies D = 6 \pd\] Thus, the pencil's height function is \[h(t) = -16t^2 + 6 \pd\] We want to find the positive value of \(t\) for which \(h(t) = 0,\) so we solve to find \[t = \sqrt{\frac{6}{16}} \approx \boxed{0.612 \un{sec}}\]
A rock is thrown straight downward at a speed of \(20\) feet per second into a canyon of depth \(180\) feet. How long does the rock take to fall to the bottom of the canyon? With what speed does the rock hit the bottom?
We want to find the rock's height function, \(h(t)\) (where \(t\) is time), to find the time when the rock hits the bottom of the canyon. We choose up to be the positive direction, so the rock's acceleration due to gravity is \(-g = -32 \undiv{ft}{sec}^2.\) We integrate this acceleration to find the rock's general velocity function: \[v(t) = \int -32 \di t = -32t + C \pd\] To solve for \(C\) we substitute the initial condition \(\orange{v(0) = -20} \col\) \[\orange{v(0)} = -32(0) + C = \orange{-20} \implies C = -20 \pd\] Hence, the rock's velocity function is \[v(t) = -32t - 20 \pd\] We then integrate \(v(t)\) to find the rock's position function, \(h(t).\) Giving this model the initial condition \(\orange{h(0) = 180}\) ensures that \(h(t)\) represents the rock's height above the canyon's bottom. We have \[h(t) = \int -32t - 20 \di t = -16t^2 - 20t + D \pd\] To solve for \(D\) we substitute the initial condition: \[\orange{h(0)} = -16(0)^2 - 20(0) + D = \orange{180} \implies D = 180 \pd\] Thus, the rock's height function is \begin{equation*} h(t) = -16t^2 - 20t + 180 \pd \end{equation*} We want the positive solution of \(h(t) = 0 \col\) \[ \ba t &= \frac{20 \pm \sqrt{(-20)^2 - 4(-16)(180)}}{2(-16)} \nl t &\approx -4.037, t \approx 2.787 \pd \ea \] Therefore, the rock takes \(2.787 \un{sec}\) to fall to the bottom of the canyon. And the rock hits the bottom with a speed of \[|v(2.787)| = |-32(2.787) - 20| = \boxed{109.184 \undiv{ft}{sec}}\]
A teddy bear is thrown from the ground straight upward with an initial speed of \(19.6\) meters per second. What is the maximum height above the ground that the teddy bear reaches?

A marshmallow is launched up into the air. After \(2\) seconds the marshmallow reaches its maximum height. Determine the marshmallow's launch velocity in feet per second.
We select the up direction to be positive. An object's acceleration as a function of time varies directly with time raised to the power of \(n,\) a positive number. With what degree of time does the object's position vary?
The object's acceleration takes the form \[a(t) = kt^n \cma\] where \(k\) is a constant. Velocity, \(v(t),\) is the integral of acceleration with respect to time; thus, \[v(t) = k \, \frac{t^{n + 1}}{n + 1} + C \pd\] If we integrate again, then the power of the \(t\) becomes \(n + 2.\) Thus, the object's position varies directly with \(t\) to the power of \(n + 2.\)
A particle moves in a straight line with initial position \(s_0,\) initial velocity \(v_0,\) and uniform acceleration \(a.\)
  1. If \(t\) is time, then show that the particle's velocity function is \[v(t) = v_0 + at \pd\]
  2. Show that the particle's position function is \[s(t) = s_0 + v_0 t + \tfrac{1}{2} at^2 \pd\]
  1. The particle's velocity function is the integral of acceleration, \(a,\) with respect to time. Since \(a\) is constant, \begin{align} v(t) &= \int a \di t \nonumber \nl &= at + C \pd \label{eq:particle-v(t)} \end{align} To solve for \(C\) we substitute in \(\eqref{eq:particle-v(t)}\) the initial condition \(\orange{v(0) = v_0} \col\) \[\orange{v(0)} = a \cdot 0 + C = \orange{v_0} \implies C = v_0 \pd\] Thus, \(\eqRefer{eq:particle-v(t)}\) becomes \begin{equation} v(t) = v_0 + at \cma \label{eq:particle-v(t)-ans} \end{equation} as requested.
  2. Position is the integral of velocity, so we integrate \(\eqref{eq:particle-v(t)-ans},\) obtaining \begin{align} s(t) &= \int v_0 + at \di t \nonumber \nl &= v_0t + \tfrac{1}{2} at^2 + D \pd \label{eq:particle-s(t)} \end{align} To solve for \(D\) we substitute in \(\eqref{eq:particle-s(t)}\) the initial condition \(\orange{s(0) = s_0} \col\) \[\orange{s(0)} = v_0(0) + \tfrac{1}{2} a(0)^2 + D = \orange{s_0} \implies D = s_0 \pd\] Thus, \(\eqRefer{eq:particle-s(t)}\) becomes \[s(t) = s_0 + v_0 t + \tfrac{1}{2} at^2 \cma\] as requested.
