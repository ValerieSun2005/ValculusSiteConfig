In Section we asserted a derivative measures a function's rate of change. If we have the rate \(f'(x)\) and want to find \(f(x),\) then we use antidifferentiation. Antidifferentiation has many applications: If we have a particle's velocity, then we can find its position at a given time. If we know an ecosystem's growth factor, then we can model the population at a given time. In this section we'll be Defining an Antiderivative; Evaluating Antiderivatives; and connecting antiderivatives to Position, Velocity, and Acceleration and Freefalling Objects.

Defining an Antiderivative

Antidifferentiation is the inverse operation of differentiation: If function \(g'\) is the derivative of \(g,\) then \(g\) is an antiderivative of \(g'.\) To generalize this principle, we say that function \(F\) is an antiderivative of \(f\) if \(F' = f.\) For example, an antiderivative of \(h(x) = x^2\) is \(H(x) = \frac{1}{3} x^3\) because \(H'(x) = x^2 = h(x).\) But \(H(x) = \frac{1}{3} x^3 + 2\) is also an antiderivative of \(f.\) Therefore, any function \(H(x) = \frac{1}{3} x^3 + C,\) where \(C\) is a constant, is an antiderivative of \(h.\) If a function has an antiderivative, then the function has infinitely many antiderivatives—all of which differ by a constant. Figure 1 shows the graphs of some antiderivatives of \(h(x) = x^2,\) in which all graphs take the form \(y = \frac{1}{3} x^3 + C\) and therefore differ by a vertical translation.

The symbol for the antidifferentiation operator is "\(\int \,\)"; for example, for \(h(x) = x^2,\) \begin{equation} H(x) = \int h(x) \di x = \int x^2 \di x = \tfrac{1}{3} x^3 + C \pd \label{eq:int-h(x)} \end{equation} The term \(\dd x\) specifies that we antidifferentiate with respect to \(x.\) Note the following terminology in \(\eqRefer{eq:int-h(x)} \col\)

  • "\(\int \,\)" is the integral sign.
  • \(h(x) = x^2\) is the integrand.
  • \(\dd x\) is the differential.
  • \(C\) is the constant of integration.
  • \(H(x) = \frac{1}{3}x^3 + C\) is an antiderivative of \(h(x).\)
  • The entire expression \(\int h(x) \di x\) is an indefinite integral.
The verbs antidifferentiate and integrate are often used interchangeably. An integral that contains bounds—for example, \(\int_1^3 x^3 \di x\)—is called a definite integral. Whereas an indefinite integral returns an antiderivative (a function), a definite integral returns a value. We will explore definite integrals in Section , in which we connect antiderivatives to calculating areas bounded by curves.

DEFINING AN ANTIDERIVATIVE
Function \(F\) is an antiderivative of \(f\) if \(F'(x) = f(x).\) Then \[\int f(x) \di x = F(x) + C \cma\] where \(C\) is a constant of integration.
When evaluating an indefinite integral, always write a constant of integration \(C\) in your answer. But when evaluating a derivative, never write a constant in your answer.

To obtain antidifferentiation statements, we read differentiation statements from right to left. For example, \[ \baat{2} &\deriv{}{x} [\sin x] = {\color{orange} \cos x} &&\Longleftrightarrow \int {\color{orange} \cos x} \di x = \sin x + C \nl &\deriv{}{x} [-\cos x] = {\color{orange} \sin x} &&\Longleftrightarrow \int {\color{orange} \sin x} \di x = -\cos x + C \nl &\deriv{}{x} [\tan x] = {\color{orange} \sec^2 x} &&\Longleftrightarrow \int {\color{orange} \sec^2 x} \di x = \tan x + C \nl &\deriv{}{x} [\sec x] = {\color{orange} \sec x \tan x} &&\Longleftrightarrow \int {\color{orange} \sec x \tan x} \di x = \sec x + C \nl &\deriv{}{x} \left[\sin^{-1} x\right] = {\color{orange}\frac{1}{\sqrt{1 - x^2}}} &&\Longleftrightarrow \int {\color{orange}\frac{1}{\sqrt{1 - x^2}}} \di x = \sin^{-1} x + C \nl &\deriv{}{x} \left[\tan^{-1} x\right] = {\color{orange}\frac{1}{1 + x^2}} &&\Longleftrightarrow \int {\color{orange}\frac{1}{1 + x^2}} \di x = \tan^{-1} x + C \nl &\deriv{}{x} \left[e^x\right] = \orange{e^x} &&\Longleftrightarrow \int \orange{e^x} \di x = e^x + C \pd \eaat \] Formally, each differentiation statement should include a constant—for example, \(\textDeriv{}{x} \, [\sin x + C] = \cos x.\) This list drops the constants in the differentiation statements for the sake of familiarity.

EXAMPLE 1
Evaluate each expression.
  1. \(\ds \int \cos x \di x\)
  2. \(\ds \int e^x \di x\)
  3. \(\ds \int \frac{1}{x} \di x\)
  1. This problem asks: "The derivative of what function is \(\cos x \ques\)" And one such function is \(\sin x.\) But the derivative of \(\sin x\) plus any constant—for example, the derivative of \((\sin x + 1)\) or of \((\sin x - 100)\)—is also \(\cos x.\) Thus, if \(C\) is any constant, then the general antiderivative of \(\cos x\) is \(\sin x + C.\) So \[\int \cos x \di x = \boxed{\sin x + C}\]
  2. The derivative of \(e^x\) is \(e^x,\) so the converse statement is that \(e^x\) is an antiderivative of \(e^x.\) We always add a constant of integration, so \[\int e^x \di x = \boxed{e^x + C}\]
  3. Since the derivative of \(\ln|x|\) is \(1/x,\) an antiderivative of \(1/x\) is \(\ln|x|.\) We then add an arbitrary constant in our final answer: \[\int \frac{1}{x} \di x = \boxed{\ln|x| + C}\]

Evaluating Antiderivatives

If \(f(x) = x^n,\) where \(n \ne -1,\) then the reverse power rule gives an antiderivative of \(f\) to be \begin{equation} F(x) = \frac{x^{n + 1}}{n + 1} \pd \label{eq:reverse-power-rule} \end{equation} In words, \(\eqRefer{eq:reverse-power-rule}\) states that to find an antiderivative of a power function (if the power isn't \(-1\)), we add one to the power and divide by the result. This fact is easy to verify because \(F'(x) = x^n = f(x).\) For example, if \(n = 4,\) then using \(\eqref{eq:reverse-power-rule}\) gives \begin{equation} \int x^4 \di x = \frac{x^{4 + 1}}{4 + 1} + C = \frac{x^5}{5} + C \pd \label{eq:int-x^4} \end{equation} We accept this answer because \(\textDeriv{}{x}\,\left[\frac{1}{5}x^5 + C\right] = x^4,\) which matches the integrand in \(\eqref{eq:int-x^4}.\) Additionally, if \(n = 0\) then \begin{equation} \int x^0 \di x = \int \dd x = \frac{x^{0 + 1}}{0 + 1} + C = x + C \pd \label{eq:int-n=0} \end{equation} In \(\int \dd x,\) the integrand is not \(0;\) rather, \(\int \dd x = \int 1 \di x.\) And \(\textDeriv{}{x}\,[x + C] = 1,\) thus affirming the result in \(\eqRefer{eq:int-n=0}.\) But if \(n = -1,\) then an antiderivative of \(f(x) = x^n\) is \(\ln|x|,\) as we asserted in (iii) of Example 1.

It is common to see expressions like \(\int \dd x/x^2,\) whose meaning is identical to \(\int \frac{1}{x^2} \di x.\) Likewise, when an integrand contains a sum or difference, using parentheses is a stylistic decision: Some people write \(\int (2x - 3)\di x,\) while others prefer \(\int 2x - 3 \di x.\) Both expressions carry the same meaning.
INTEGRATING POWER FUNCTIONS
If \(n \ne -1,\) then the reverse power rule gives \[\int x^n \di x = \frac{x^{n + 1}}{n + 1} + C \pd \] If \(n = -1,\) then \(\int (1/x) \di x = \ln|x| + C.\)

If functions \(f\) and \(g\) have antiderivatives \(F\) and \(G,\) respectively, then \begin{equation} \int f(x) \pm g(x) \di x = \int f(x) \di x \pm \int g(x) \di x = F(x) \pm G(x) + C \pd \label{eq:int-sum} \end{equation} If \(k\) is a constant, then \begin{equation} \int kf(x) \di x = k \int f(x) \di x = kF(x) + C \pd \label{eq:int-constant} \end{equation} Antiderivatives are said to be linear because they satisfy \(\eqref{eq:int-sum}\) and \(\eqref{eq:int-constant}.\) In \(\eqRefer{eq:int-sum},\) \(\int f(x) \di x\) and \(\int g(x) \di x\) each output a constant of integration; we write the sum or difference of these constants simply as \(C,\) because adding or subtracting two constants gives another arbitrary constant. By similar logic, multiplying or dividing a constant of integration by a number does not alter the identity of the constant—the constant can still be any value. For example, if \(C\) is a constant of integration, then we may write expressions such as \(2C\) and \(C + 100\) simply as \(C.\)

LINEARITY OF ANTIDERIVATIVES
  1. \(\int f(x) \pm g(x) \di x = \int f(x) \di x \pm \int g(x) \di x.\)
  2. If \(k\) is a constant, then \(\int kf(x) \di x = k \int f(x) \di x.\)

We then pull out the constants in the second and third integrals. Doing so gives \begin{align*} \int t^{1/2} \di t - 2 \int \sin t \di t + 4 \int \frac{1}{1 + t^2} \di t &= \frac{t^{1 + 1/2}}{1 + 1/2} -2(-\cos t) + 4 \tan^{-1} t + C \nl &= \boxed{\tfrac{2}{3} t^{3/2} + 2 \cos t + 4 \tan^{-1} t + C} \end{align*}

An equation that relates \(f\) and its derivatives is called a differential equation. In calculus, we often have a rate \(f'\) and need to find a solution \(f.\) (Differential equations are fundamental to modeling natural phenomena, which we discuss further in Section .) We often want \(f\) that satisfies a specific point, called an initial condition. To find \(f\) given \(f'\) and an initial condition, we first integrate \(f'.\) Doing so produces a general solution that contains a constant of integration. We then plug in the initial condition to solve for this constant. For example, if \(f'(x) = 2x\) and \(\orange{f(1) = 2},\) then we antidifferentiate \(f'(x)\) and substitute the given initial condition to solve for the constant of integration: \begin{align} f(x) = \int 2x \di x &= x^2 + C \nonumber \nl \orange{f(1)} &= (1)^2 + C = \orange{2} \implies C = 1 \pd \nonumber \nl \therefore f(x) &= \boxed{x^2 + 1} \label{eq:diff-eq-f(x)-particular} \end{align} \(\eqRefer{eq:diff-eq-f(x)-particular}\) is called the particular solution of the differential equation \(f'(x) = 2x\) that satisfies the initial condition \(\orange{f(1) = 2}.\)

EXAMPLE 3
Suppose \(f'(x) = e^x + 2x + 7\) and \(f(0) = 4.\) Find \(f(x).\)

We antidifferentiate \(f',\) finding \begin{equation} f(x) = e^x + x^2 + 7x + C \pd \label{eq:ex-diff-eq} \end{equation} We call \(\eqRefer{eq:ex-diff-eq}\) a general solution to the differential equation \(f'(x) = e^x + 2x + 7.\) To find the particular solution, we substitute in \(\eqref{eq:ex-diff-eq}\) the given initial condition, \(\orange{f(0) = 4},\) to obtain \[\orange{f(0)} = e^0 + (0)^2 + 7(0) + C = \orange 4 \implies C = 3 \pd\] Thus, substituting \(C = 3\) in \(\eqref{eq:ex-diff-eq}\) gives \begin{equation} f(x) = \boxed{e^x + x^2 + 7x + 3}\cma \label{eq:ex-diff-eq-f(x)-particular} \end{equation} the particular solution that passes through the point \((0, 4).\)

Checking our answer is simple: It's easy to see that \(\eqRefer{eq:ex-diff-eq-f(x)-particular}\) passes through \((0, 4)\) because \[\orange{f(0)} = e^0 + 0^2 + 7(0) + 3 \equalsCheck \orange{4} \pd\] Moreover, differentiating \(\eqref{eq:ex-diff-eq-f(x)-particular}\) yields \[f'(x) = e^x + 2x + 7 \cma\] which matches the differential equation we were given.

EXAMPLE 4
In Figure 2, the graph of \(y = f(x)\) is shown for \(0 \lt x \lt 4.\) Let \(F\) be an antiderivative of \(f\) such that \(F(0) = 2.\) Sketch a rough graph of \(y = F(x).\)
The key idea to sketching \(y = F(x)\) is that at any point \(x,\) the slope of \(F(x)\) is \(f(x).\) In Figure 3 we begin the graph at the point \((0, 2).\) We make the following remarks about the first derivative of \(F,\) \(f \col\)
  • When \(0 \lt x \lt 1\) and \(x \gt 3,\) \(f\) is negative, meaning the slope of \(F\) is negative. Thus, \(F\) is decreasing on these intervals.
  • For \(1 \lt x \lt 3,\) \(f\) is positive, so the slope of \(F\) is positive. Therefore, \(F\) is increasing on this interval.
  • Since \(f(1) = 0\) and \(f(3) = 0,\) \(F\) has horizontal tangents when \(x = 1\) and \(x = 3.\) By the First Derivative Test (see Section ), \(F(1)\) is a local minimum because \(F' = f\) changes sign from negative to positive at \(x = 1,\) and \(F(3)\) is a local maximum because \(F' = f\) changes sign from positive to negative at \(x = 3.\)
We now consider the graph's shape using the second derivative of \(F,\) \(f' \col\)
  • \(F''(2) = f'(2) = 0,\) so \(F\) has an inflection point when \(x = 2.\)
  • \(F'' = f' \gt 0\) for \(0 \lt x \lt 2,\) so \(F\) is concave up on this interval. In contrast, \(F'' = f' \lt 0\) when \(x \gt 2,\) so \(F\) is concave down for \(x \gt 2.\)

Position, Velocity, and Acceleration

In Section we connected position to velocity and acceleration using derivatives. These relationships are examples of differential equations. We now relate these quantities using integrals, and we have the tools to solve these differential equations. If \(s(t)\) represents an object's position at time \(t,\) then the particle's velocity function is \(v(t) = \textDeriv{s}{t}.\) Thus, the reverse statement is that \(s(t)\) is the integral of \(v(t)\) with respect to time: \begin{equation} s(t) = \int v(t) \di t \pd \label{eq:s(t)=intv(t)} \end{equation} Acceleration, \(a(t),\) is the derivative of velocity, so velocity is the integral of acceleration: \begin{equation} v(t) = \int a(t) \di t \pd \label{eq:v(t)=inta(t)} \end{equation}

EXAMPLE 5
A particle travels along the \(x\)-axis with acceleration given by \(a(t) = 4t + 6,\) where \(t\) is measured in seconds and \(x\) is measured in inches. Given that the particle's initial position is \(x(0) = 2\) and its initial velocity is \(v(0) = 3,\) find its position function \(x(t).\)

Velocity is the integral of acceleration [\(\eqRefer{eq:v(t)=inta(t)}\)]. The particle's general velocity function is then \begin{align} v(t) &= \int 4t + 6 \di t \nonumber \nl &= 2t^2 + 6t + C \cma \label{eq:particle-xaxis-v(t)} \end{align} where \(C\) is a constant of integration. To find the particle's particular velocity function, in which \(C\) has a value, we note that the particle's initial velocity is \(\orange{v(0) = 3}.\) Substituting this point in \(\eqref{eq:particle-xaxis-v(t)}\) gives \[\orange{v(0)} = 2(0)^2 + 6(0) + C = \orange 3 \implies C = 3 \pd\] Thus, \(\eqref{eq:particle-xaxis-v(t)}\) becomes \begin{equation} v(t) = 2t^2 + 6t + 3 \pd \label{eq:particle-xaxis-v(t)-spec} \end{equation}

Position is the integral of velocity [\(\eqref{eq:s(t)=intv(t)}\)], so we obtain the general position function by integrating \(\eqref{eq:particle-xaxis-v(t)-spec} \col\) \begin{align} x(t) &= \int 2t^2 + 6t + 3 \di t \nonumber \nl &= \tfrac{2}{3} t^3 + 3t^2 + 3t + D \cma \label{eq:particle-xaxis-x(t)} \end{align} where \(D\) is a constant. To solve for \(D\) we plug into \(\eqref{eq:particle-xaxis-x(t)}\) the initial condition \(\orange{x(0) = 2}\) to obtain \[\orange{x(0)} = \tfrac{2}{3}(0)^3 + 3(0)^2 + 3(0) + D = \orange 2 \implies D = 2 \pd\] Thus, \(\eqRefer{eq:particle-xaxis-x(t)}\) becomes \begin{equation} x(t) = \boxed{\tfrac{2}{3} t^3 + 3t^2 + 3t + 2} \label{eq:particle-xaxis-x(t)-particular} \end{equation}

To check our answer, we need to verify that \(\eqRefer{eq:particle-xaxis-x(t)-particular}\) satisfies the given initial conditions and that \(x''(t) = 4t + 6,\) the given differential equation. It's easy to see that \(\eqref{eq:particle-xaxis-x(t)-particular}\) satisfies \(\orange{x(0) = 2} \col\) \[\orange{x(0)} = \tfrac{2}{3}(0)^3 + 3(0)^2 + 3(0) + 2 \equalsCheck \orange{2} \pd\] Moreover, differentiating \(\eqref{eq:particle-xaxis-x(t)-particular}\) shows \begin{equation} x'(t) = v(t) = 2t^2 + 6t + 3 \pd \label{eq:particle-xaxis-x(t)-particular-diff} \end{equation} This expression should satisfy the given \(\orange{v(0) = 3} \col\) \[\orange{v(0)} = 2(0)^2 + 6(0) + 3 \equalsCheck \orange{3} \pd\] Finally, we differentiate \(\eqref{eq:particle-xaxis-x(t)-particular-diff} \col\) \[x''(t) = a(t) \equalsCheck 4t + 6 \cma\] which is the expression given.

Freefalling Objects

On Earth's surface, the acceleration due to gravity is \(g,\) whose value is \(32 \undiv{ft}{sec}^2\) or \(9.8 \undiv{m}{sec}^2.\) When objects fall, we select a direction to be positive and a direction to be negative. It's intuitive to assign the positive direction to be upward, and let the object's position function represent its height above the ground. For example, when an object is thrown upward, its initial velocity is positive and its acceleration is \(-g.\) Conversely, as an object falls to the ground, its velocity is always negative and its acceleration is \(-g.\) Let's make a few remarks about freefalling objects:

  • Remark 1. An object dropped from rest has an initial velocity of \(0.\)
  • Remark 2. When an object hits the ground, its final velocity is not \(0.\) We take the object's final velocity to be the velocity the instant before it hits the ground.
  • Remark 3. An object that is thrown upward reaches its maximum height when its velocity is \(0.\)

EXAMPLE 6
A rock is dropped from the top of \(100\)-foot cliff. Derive an expression for the rock's height as a function of time, \(t,\) in seconds. How long does the rock take to fall to the ground? What is the rock's speed right before it hits the ground?

Since the rock is dropped from rest, its initial velocity is \(0\) (Remark 1). Its initial position is \(100 \un{ft}.\) We take the positive direction to be upward; the rock's velocity and acceleration are therefore both negative as it falls.

By \(\eqRefer{eq:v(t)=inta(t)},\) the rock's velocity function is the integral of acceleration. The acceleration due to gravity is \(g = -32 \undiv{ft}{sec}^2,\) so we get \begin{align} v(t) &= \int -32 \di t \nonumber \nl &= -32t + C \cma \label{eq:rock-v(t)-general} \end{align} where \(C\) is a constant of integration. To solve for \(C\) we substitute in \(\eqref{eq:rock-v(t)-general}\) the initial condition \(\orange{v(0) = 0} \col\) \[\orange{v(0)} = -32(0) + C = \orange 0 \implies C = 0 \pd\] Thus, the rock's velocity function is \begin{equation} v(t) = -32t \pd \label{eq:rock-v(t)-particular} \end{equation}

To obtain position we integrate \(\eqref{eq:rock-v(t)-particular}\) to get \begin{equation} h(t) = -16t^2 + D \cma \label{eq:rock-h(t)-general} \end{equation} where \(D\) is a constant. Function \(h(t)\) represents the rock's height above the ground, so this model has the initial condition \(\orange{h(0) = 100}.\) In \(\eqref{eq:rock-h(t)-general}\) we substitute this initial condition to solve for \(D \col\) \[\orange{h(0)} = -16(0)^2 + D = \orange{100} \implies D = 100 \pd\] Thus, \(\eqRefer{eq:rock-h(t)-general}\) becomes \begin{equation} h(t) = \boxed{-16t^2 + 100} \label{eq:rock-h(t)-ans} \end{equation}

The rock's final velocity is not \(0\) (Remark 2); it is instead the rock's velocity right before it hits the ground—when \(h(t) = 0.\) Equating \(\eqref{eq:rock-h(t)-ans}\) to \(0\) and solving for the positive value of \(t,\) we have \[-16t^2 + 100 = 0 \implies t = \tfrac{10}{4} = 2.5 \pd\] The ball therefore hits the ground after \(2.5\) seconds. In \(\eqRefer{eq:rock-v(t)-particular}\) we substitute \(t = 2.5 \col\) \[v(t) = -32(2.5) = -80 \pd\] Speed is the magnitude (absolute value) of velocity, so the rock's final speed is \(|-80| = 80 \undiv{ft}{sec}.\)

EXAMPLE 7
A ball is launched from the ground straight upward at a speed of \(16\) feet per second.
  1. Find the ball's velocity function. After how many seconds does the ball reach its maximum height?
  2. How high does the ball travel?
  1. We choose up to be the positive direction. The ball's initial velocity is \(16 \undiv{ft}{sec},\) and the ball's acceleration due to gravity is \(-32 \undiv{ft}{sec}^2.\) By \(\eqRefer{eq:v(t)=inta(t)},\) we integrate acceleration to find the ball's general velocity function: \begin{align} v(t) &= \int -32 \di t \nonumber \nl &= -32 t + C \pd \label{eq:ball-v(t)} \end{align} We solve for \(C\) by substituting in \(\eqref{eq:ball-v(t)}\) the initial condition \(\orange{v(0) = 16}\col\) \[\orange{v(0)} = -32(0) + C = \orange{16} \implies C = 16 \pd\] Thus, \(\eqref{eq:ball-v(t)}\) becomes \begin{equation} v(t) = \boxed{-32t + 16} \label{eq:ball-v(t)-ans} \end{equation} The ball reaches its maximum height when \(v = 0\) (Remark 3). In \(\eqRefer{eq:ball-v(t)-ans}\) solving \(v(t) = 0\) gives \(t = 16/32.\) The ball therefore reaches its maximum height after \(0.5 \un{sec}.\)
  2. We need the object's height function, \(h(t).\) Integrating velocity in \(\eqref{eq:ball-v(t)-ans}\) gives \begin{equation} h(t) = -16t^2 + 16t + D \cma \label{eq:ball-h(t)} \end{equation} where \(D\) is a constant of integration. We choose the ground to be \(h = 0;\) the ball initially begins on the ground, so \(\orange{h(0) = 0}.\) Plugging this initial condition into \(\eqRefer{eq:ball-h(t)}\) shows \[\orange{h(0)} = -16(0)^2 + 16(0) + D = \orange 0 \implies D = 0 \pd\] Thus, \(\eqref{eq:ball-h(t)}\) becomes \[h(t) = -16t^2 + 16t \pd\] The ball therefore reaches a height of \[s \par{\tfrac{1}{2}} = -16 \par{\tfrac{1}{2}}^2 + 16 \par{\tfrac{1}{2}} = \boxed{4 \un{ft}}\]

Antidifferentiation is the inverse operation of differentiation. Function \(F\) is called an antiderivative of function \(f\) if \(F' = f.\) We represent an antiderivative by using an indefinite integral: \[\int f(x) \di x = F(x) + C \cma\] where "\(\int \,\)" is the integral sign, \(f(x)\) is the integrand, \(\dd x\) is the differential, \(F(x)\) is an antiderivative of \(f(x),\) and \(C\) is the constant of integration. If a function has an antiderivative, then the function has infinitely many antiderivatives—all of which differ by a constant, \(C.\) The verbs antidifferentiate and integrate are interchangeable. It is easy to verify your answer to an integral: differentiating your answer should yield the integrand you were given. Always write \(+C\) when you evaluate an indefinite integral, but never write \(+C\) when you evaluate a derivative.

To integrate a power function \(x^n,\) \(n \ne -1,\) we use the reverse power rule: \[\int x^n \di x = \frac{x^{n + 1}}{n + 1} + C \pd\] But if \(n = -1,\) then \(\int x^{-1} \di x = \ln|x| + C.\)

Antiderivatives are linear; that is, if \(f\) and \(g\) are functions and \(k\) is a constant, then

  • \(\int f(x) \pm g(x) \di x = \int f(x) \di x \pm \int g(x) \di x.\)
  • If \(k\) is a constant, then \(\int kf(x) \di x = k \int f(x) \di x.\)

A differential equation relates function \(f\) with its derivatives. To solve a differential equation in which you are given \(f'\) and a specific point (called an initial condition), integrate \(f'\) and plug in the initial condition to solve for the constant of integration \(C.\) A general solution of a differential equation does not have a value for the constant \(C,\) whereas a particular solution does have a value for \(C.\) If you are given an initial condition, then you want to find the particular solution.

In particle motion, position is related to velocity and acceleration through differential equations. Velocity and acceleration are then connected to position by integrals: If \(t\) is time, \(s(t)\) is position, \(v(t)\) is velocity, and \(a(t)\) is acceleration, then

  • \(v(t) = \int a(t) \di t.\)
  • \(s(t) = \int v(t) \di t.\)

To solve problems with freefalling objects, select an axis orientation. It is logical to let the upward direction be the positive direction. In these cases a falling object's acceleration is \(-9.8 \undiv{m}{sec}^2,\) or \(-32 \undiv{ft}{sec}^2.\) Note the following:

  • Remark 1. An object dropped from rest has an initial velocity of \(0.\)
  • Remark 2. When an object hits the ground, its final velocity is not \(0.\) We take the object's final velocity to be the velocity the instant before it hits the ground.
  • Remark 3. An object that is thrown upward reaches its maximum height when its velocity is \(0.\)
