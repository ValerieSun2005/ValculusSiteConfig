In the world, many phenomena require efficiency. Therefore, locating the extrema—the minima or maxima—of a model is a fundamental skill of calculus. In this section, we define the forms of extrema: We begin with Absolute Extrema, which we define using the Extreme Value Theorem. Then we examine Critical Points to define Relative Extrema. This section culminates in the Closed Interval Method, a problem-solving strategy for finding extrema.

Absolute Extrema

Figure 1 shows the graph of \(y = g(x).\) We say \(g(1) = 8/3\) is the absolute minimum of \(g\) because \((1, 8/3)\) is the lowest point on the graph. Conversely, \(g(3) = 4\) is the absolute maximum of \(g\) because \((3, 4)\) is the highest point on the graph. Each of these values is an absolute extremum (plural, extrema) of \(g.\) To formally define these terms, consider \(f(x)\) for \(a \leq x \leq b.\) Let \(c\) be in \([a, b].\)

  • \(f(c)\) is the absolute maximum value of \(f\) if \(f(c) \geq f(x)\) for all \(a \leq x \leq b.\)
  • \(f(c)\) is the absolute minimum value of \(f\) if \(f(c) \leq f(x)\) for all \(a \leq x \leq b.\)
We may also call \(f(c)\) the global maximum or global minimum, respectively.

ABSOLUTE EXTREMA
Consider \(f(x)\) for \(a \leq x \leq b.\) Suppose \(c\) is in \([a, b].\)
  • \(f(c)\) is the absolute maximum value of \(f\) if \(f(c) \geq f(x)\) for all \(a \leq x \leq b.\)
  • \(f(c)\) is the absolute minimum value of \(f\) if \(f(c) \leq f(x)\) for all \(a \leq x \leq b.\)

As shown by Figure 2, the absolute minimum of \(g(x) = x^2\) is \(g(0) = 0.\) But \(g\) has no maximum because the graph of \(y = x^2\) has no highest value. Similarly, \(h(x) = x^3\) has no extrema, since its graph has no lowest point or highest point (Figure 3). Thus, we conclude that a function may not have an absolute minimum or absolute maximum over its entire domain. Over a closed interval, however, a continuous function must have both an absolute minimum and absolute maximum. This is by the Extreme Value Theorem.

EXTREME VALUE THEOREM
If \(f\) is continuous over \([a, b],\) then \(f\) attains an absolute minimum value \(f(c)\) and an absolute maximum value \(f(d),\) where \(c\) and \(d\) are in \([a, b].\)

For example, the maximum value of \(g(x) = x^2\) in \(0 \leq x \leq 2\) is \(g(2) = 4,\) while the minimum value in this interval is \(g(0) = 0.\) Similarly, \(h(x) = x^3\) for \(-1 \leq x \leq 1\) has a maximum of \(h(1) = 1\) and a minimum of \(h(-1) = -1.\) In each of these cases, the extrema lie at the endpoints of a closed interval. By the Extreme Value Theorem, an extreme value can be attained multiple times. For example, the graph of \(y = \sin x\) has infinitely many peaks and troughs—corresponding to the absolute maxima and absolute minima, respectively. If \(f\) is uniform, then the minimum value and maximum value are the same. For example, the absolute minimum and absolute maximum of \(g(x) = 5\) are both \(5.\)

The derivative of \(f(x) = 1/x\) is \(f'(x) = -1/x^2,\) which is undefined at \(x = 0.\) But since \(f(0)\) is also undefined, \(x = 0\) does not correspond to a critical point of \(f.\)

Critical Points

A function \(f\) has a critical point when \(x = c\) if \(f'(c) = 0\) or if \(f'(c)\) is undefined and \(f(c)\) is defined. We then call \(c\) a critical number of \(f.\) For example, we find the critical numbers of \(g(x) = \frac{1}{3} x^3 - 2x^2 + 3x - 1\) by setting \(g'(x) = 0.\) Since \(g\) is a polynomial, its derivative is never undefined. Thus, we find \[ \ba g'(x) = x^2 - 4x + 3 &= 0 \nl (x - 3)(x - 1) &= 0 \comma \ea \] whose solutions are \(x = 1\) and \(x = 3.\) So these are the critical numbers of \(g.\)

EXAMPLE 1
For each of (i)–(iii), determine the critical numbers of the given function \(g.\)
  1. \(g(x) = \frac{1}{4}x^4 - \frac{4}{3}x^3 + \frac{3}{2}x^2 + 3\)
  2. \(g(x) = \sqrt[3]{x - 3}\)
  3. \(g(t) = \sqrt[3]{t} \, (4 - t)\)
  1. The function \(g(x)\) has a critical point when \(x = c\) if \(g'(c) = 0\) or if \(g'(c)\) is undefined and \(g(c)\) is defined. Since \(g(x)\) is a polynomial, it is continuous and differentiable for all \(x.\) Therefore, \(g'(x)\) is never undefined; instead, we need \(g'(x) = 0.\) Thus, we solve for \(x\) in \(g'(x) = 0 \col\) \[ \ba g'(x) = x^3 - 4x^2 + 3x &= 0 \nl x(x - 3)(x - 1) &= 0 \nl x &= 0 \comma x = 1 \comma x = 3 \pd \ea \] Therefore, the critical numbers of \(g(x)\) are \(x = 0,\) \(x = 1,\) and \(x = 3.\)
  2. The derivative of \(g\) is \[g'(x) = \frac{1}{3(x - 3)^{2/3}} \comma\] which is undefined at \(x = 3.\) Because \(g(3)\) is defined and \(g'(3)\) is undefined, \(x = 3\) is a critical number of \(g.\)
  3. We differentiate \(g\) to find \[g'(t) = \frac{4 - t}{3t^{2/3}} - \sqrt[3]{t} \pd\] Solving \(g'(t) = 0\) gives \(t = 1.\) Furthermore, \(g'(0)\) is undefined but \(g(0)\) is defined. Therefore, \(t = 0\) and \(t = 1\) are critical numbers of \(g.\)

Relative Extrema

We now discuss how to find the relative minima and relative maxima of a function \(f.\) (These terms are also called local extrema.) Graphically, these points represent troughs and peaks, respectively, of the graph of \(y = f(x).\) In Figure 4, Point \(A\) is the location of a relative minimum of \(f,\) whereas Point \(B\) is the location of a relative maximum of \(f.\)

Points \(A\) and \(B\) are both critical points because \(f' = 0\) at each point. At Point \(A\) the sign of \(f'\) flips from negative to positive. Conversely, at Point \(B,\) \(f'\) changes sign from positive to negative. We then see the following: If \(x = c\) is a critical number of \(f(x),\) then

  • \(f\) has a relative minimum at \(c\) if \(f'\) changes sign from negative to positive at \(c.\)
  • \(f\) has a relative maximum at \(c\) if \(f'\) changes sign from positive to negative at \(c.\)
  • \(f\) has neither a relative minimum nor relative maximum at \(c\) if \(f'\) does not change sign at \(c.\)
We call these guidelines the First Derivative Test. For example, the derivative of \(g(x) = x^3\) is \(g'(x) = 3x^2,\) satisfying \(g'(0) = 0.\) Therefore, \(x = 0\) is a critical number of \(g.\) But \(g'(x) = 3x^2 \geq 0,\) so \(g'\) does not change sign at \(x = 0.\) Hence, \(x = 0\) is not a relative extremum of \(g\) (see Figure 3).

FIRST DERIVATIVE TEST
If \(c\) is a critical number of \(f,\) then
  1. \(f\) has a relative minimum at \(c\) if \(f'\) changes sign from negative to positive at \(c.\)
  2. \(f\) has a relative maximum at \(c\) if \(f'\) changes sign from positive to negative at \(c.\)
  3. \(f\) has neither a relative minimum nor relative maximum at \(c\) if \(f'\) does not change sign at \(c.\)
EXAMPLE 2
In Figure 5, classify what type of extremum \(f\) has at points \(A,\) \(B,\) and \(C.\)
Point \(A\) is the location of a relative minimum of \(f.\) Point \(B\) is the location of a relative maximum of \(f.\) Point \(C\) is the location of a relative minimum and the absolute minimum of \(f.\)
EXAMPLE 3
Let \(g(x) = 2x^3 - 3x^2 - 1.\) Find all the relative extrema of \(g.\)
By the First Derivative Test, a critical number \(c\) is a relative extremum of \(g\) if \(g'\) changes sign at \(c.\) We find the critical numbers of \(g\) by letting \(g'(x) = 0 \col\) \[ \ba g'(x) = 6x^2 - 6x &= 0 \nl 6x(x - 1) &= 0 \nl \implies x &= 0 \comma x = 1 \pd \ea \] Now we analyze the sign changes of \(g'\) at each of these critical numbers. In so doing, it's helpful to use the factored form, \(g'(x) = 6x(x - 1).\)
  • For \(x \lt 0,\) \(g' \gt 0;\) for \(0 \lt x \lt 1,\) \(g' \lt 0.\) Therefore, because \(g'\) changes sign from positive to negative at \(x = 0,\) the First Derivative Test asserts that \(g(x)\) has a relative maximum when \(x = 0.\) We conclude that \(g(0) = -1\) is a relative maximum of \(g.\)
  • For \(0 \lt x \lt 1,\) \(g' \lt 0;\) for \(x \gt 1,\) \(g' \gt 0.\) The First Derivative Test then states that \(x = 1\) corresponds to a relative minimum of \(g.\) We find \(g(1) = -2,\) a relative minimum of \(g.\)
EXAMPLE 4
Let \(g(x)\) be a continuous function for \(-6 \leq x \leq 6.\) The graph of \(y = g'(x)\) is shown in Figure 6. Find the critical numbers of \(g.\) At each critical number, determine whether \(g\) has a relative minimum, relative maximum, or neither.
 

The function \(g\) has a critical number at \(c\) if \(g'(c) = 0\) or if \(g(c)\) exists but \(g'(c)\) is undefined. From Figure 6, \(g'(x) = 0\) when \(x = -4\) and \(x = 5.\) Moreover, \(g'(x)\) is undefined when \(x = 1\) and \(x = 3,\) whereas \(g(1)\) and \(g(3)\) both exist because it is given that \(g\) is continuous on \([-6, 6].\) Therefore, the critical numbers of \(g\) are \(x = -4,\) \(x = 1,\) \(x = 3,\) and \(x = 5.\) Note that \(x = -6\) and \(x = 6\) are the endpoints of \(g,\) so they are not critical numbers.

At \(x = -4,\) \(g'\) changes sign from negative to positive. The First Derivative Test then states that \(g\) has a relative minimum at \(x = -4.\) Likewise, at \(x = 1\) the sign of \(g'\) switches from positive to negative, so \(g\) has a relative maximum at \(x = 1.\) Conversely, at \(x = 3\) and \(x = 5,\) \(g'\) does not change sign. Thus, these critical numbers do not correspond to relative minima or relative maxima.

We now present a coherent strategy for locating absolute extrema. If \(f\) is continuous over \([a, b],\) then the Extreme Value Theorem guarantees that \(f\) has both an absolute minimum and absolute maximum in \([a, b].\) We deduce the following: If \(f\) is continuous over \([a, b]\) and has a critical number \(c\) in \((a, b),\) then the absolute minimum or absolute maximum of \(f\) must lie at the critical number \(c\) or at the endpoints \(a\) and \(b.\) In other words, an extreme value must be at a critical point or an endpoint. To find the absolute minimum and absolute maximum of \(f,\) we use the Closed Interval Method, shown below.

CLOSED INTERVAL METHOD
Suppose \(f\) is continuous over \([a, b].\) To find the absolute minimum and absolute maximum of \(f,\) use the following procedure.
  1. Evaluate \(f\) at the critical numbers in \((a, b).\)
  2. Evaluate \(f\) at the endpoints \(a\) and \(b.\)
  3. Determine the smallest and largest of the values in Steps 1 and 2. The largest value is the absolute maximum of \(f,\) while the smallest value is the absolute minimum of \(f.\)
EXAMPLE 5
Let \(g(x) = x^3 - \frac{3}{2} x^2 - 6x.\) Find the absolute minimum and absolute maximum of \(g\) over the interval \(-2 \leq x \leq 4.\)

The Closed Interval Method asserts that an absolute extremum of \(g\) must either lie at the endpoints of \([-2, 4]\) or at a critical number in \((-2, 4).\) We differentiate \(g\) to determine its critical numbers: \[ \ba g'(x) = 3x^2 - 3x - 6 &= 0 \nl \implies x &= -1 \comma x = 2 \pd \ea \]

We now evaluate \(g\) at these two critical numbers and at the endpoints. The values of \(g\) at the critical points are \[g(-1) = 7/2 \and g(2) = -10 \pd\] Additionally, the values of \(g\) at the endpoints of \([-2, 4]\) are \[g(-2) = -2 \and g(4) = 16 \pd\] The smallest of these values is \(g(2) = -10,\) and the largest is \(g(4) = 16.\) Thus, the minimum value of \(g\) in \([-2, 4]\) is \(-10,\) and the maximum value is \(16.\)

A function \(f\) may have two types of extrema: absolute extrema and relative extrema. Consider \(f(x)\) for \(a \leq x \leq b.\) Suppose \(c\) is in \([a, b].\) We call \(f(c)\) the absolute maximum value of \(f\) if \(f(c) \geq f(x)\) for all \(a \leq x \leq b.\) In contrast, \(f(c)\) is the absolute minimum value of \(f\) if \(f(c) \leq f(x)\) for all \(a \leq x \leq b.\)

The Extreme Value Theorem states the following: if \(f\) is continuous over \([a, b],\) then \(f\) attains an absolute minimum value \(f(c)\) and an absolute maximum value \(f(d),\) where \(c\) and \(d\) are in \([a, b].\)

A function \(f(x)\) has a critical point at \(c\) if \(f'(c) = 0\) or if \(f'(c)\) is undefined and \(f(c)\) is defined. Then \(x = c\) is called a critical number of \(f(x).\)

The First Derivative Test states the following: If \(c\) is a critical number of \(f,\) then

  1. \(f\) has a relative minimum at \(c\) if \(f'\) changes sign from negative to positive at \(c.\)
  2. \(f\) has a relative maximum at \(c\) if \(f'\) changes sign from positive to negative at \(c.\)
  3. \(f\) has neither a relative minimum nor relative maximum at \(c\) if \(f'\) does not change sign at \(c.\)

We use the Closed Interval Method to determine the absolute minimum and absolute maximum of a function \(f\) over a closed interval \([a, b].\) The procedure is as follows.

  1. Evaluate \(f\) at the critical numbers in \((a, b).\)
  2. Evaluate \(f\) at the endpoints \(a\) and \(b.\)
  3. Determine the smallest and largest of the values in Steps 1 and 2. The largest value is the absolute maximum of \(f,\) while the smallest value is the absolute minimum of \(f.\)
