Motivation

Consider the family of integrals \(\int \sqrt{a^2 - x^2} \, \dd x,\) \(a \gt 0,\) which arise when finding the area under a circle or ellipse. To integrate this form, our plan should be to remove this root sign. Let's substitute \(x = a \sin \theta.\) Then, using the identity \(1 - \sin^2 \theta = \cos^2 \theta,\) the integrand becomes \[\sqrt{a^2 - x^2} = \sqrt{a^2 - a^2 \sin \theta} = a \sqrt{1 - \sin^2 \theta} = a \abs{\cos \theta} \period\] This substitution eliminates the root sign. Because \(x = a \sin \theta,\) we have \(\dd x = a \cos \theta \, \dd \theta.\) In an indefinite integral, we drop the absolute value bars for simplicity. Thus, our integral is \[ \int \sqrt{a^2 - x^2} \, \dd x = \int a^2 \cos^2 \theta \, \dd \theta \comma \] which we evaluate using the reduction formula \(\cos^2 \theta = \frac{1}{2}(1 + \cos 2 \theta).\) We call this substitution procedure trigonometric substitution: the procedure by which we exploit trig identities to strip root signs.

Substitutions

We've established that for the expression \(\sqrt{a^2 - x^2},\) which we'll denote Case 1, we should let \(x = a \sin \theta.\) But let's consider two more cases: Case 2, \(\sqrt{x^2 - a^2};\) and Case 3, \(\sqrt{a^2 + x^2}.\) For Case 2, we let \(x = a \sec \theta.\) Using the identity \(\sec^2 \theta - 1 = \tan^2 \theta,\) we have \[\sqrt{x^2 - a^2} = \sqrt{a^2 \sec^2 \theta - a^2} = a \abs{\tan \theta} \period\] Similarly, for Case 3 we substitute \(x = a \tan \theta.\) We then use the identity \(\tan^2 \theta + 1 = \sec^2 \theta\) to get \[\sqrt{a^2 + x^2} = \sqrt{a^2 + a^2 \tan^2 \theta} = a \abs{\sec \theta} \period\]

Note that in Case 1 we can also substitute \(x = a \cos \theta,\) which simplifies the radical to \(a \abs{\sin \theta}.\) This substitution accomplishes the same goal, eliminating the root; but the differential, \(\dd x = -a \sin \theta \, \dd \theta,\) has an extra negative sign. Similarly, using \(x = a \csc \theta\) for Case 2 and \(x = a \cot \theta\) for Case 3 both eliminate root signs, but their differentials contain negative signs. These substitutions are therefore more complicated, so we avoid them.

We need to ensure that each substitution we make is one-to-one; that is, it has an inverse function. For example, when substituting \(x = a \sin \theta,\) \(x\) has an inverse function for \(- \pi/2 \leq \theta \leq \pi/2.\) Thus, we restrict \(\theta\) to the interval \([- \pi/2, \pi/2].\) Note that this interval matches the range of inverse sine. Conversely, in Case 2 we substitute \(x = a \sec \theta.\) Here, \(x\) is one-to-one for \(- \pi/2 \lt \theta \lt \pi/2,\) so we restrict \(\theta\) to this interval. Finally, Case 3 presents the substitution \(x = a \tan \theta,\) which has an inverse function for either \(0 \leq \theta \lt \pi/2\) or \(\pi \leq \theta \lt 3 \pi/2.\)

EXAMPLE 1
Evaluate \(\ds \int \frac{1}{x \sqrt{4 - x^2}} \, \dd x\) using trigonometric substitution.

The denominator has the factor \(\sqrt{4 - x^2},\) whose root we can clear using \[ \ba x &= 2 \sin \theta \\ \dd x &= 2 \cos \theta \, \dd \theta \period \ea \] The integral then becomes \[ \ba \int \frac{1}{(2 \sin \theta) \sqrt{4 - 4 \sin^2 \theta}} \cdot 2 \cos \theta \, \dd \theta &= \int \frac{1}{2} \csc \theta \, \dd \theta \period \ea \] Recall that \(\int \csc \theta \, \dd \theta = - \ln \abs{\csc \theta + \cot \theta} + C.\) Thus, we have \[\int \frac{1}{2} \csc \theta \, \dd \theta = -\frac{1}{2} \ln \abs{\csc \theta + \cot \theta} + C \period\]

At this stage, we need to convert back to \(x.\) Since \(x = 2 \sin \theta,\) it follows that \(\sin \theta = x/2.\) We can then draw a right triangle that satisfies this description: The triangle has a hypotenuse of length \(2\) and an angle \(\theta\) whose opposite side is \(x\) (Figure 1). We then see that \(\csc \theta = 2/x\) and \(\cot \theta = \sqrt{4 - x^2}/x.\) Therefore, our final answer is \[\boxed{-\frac{1}{2} \ln \abs{\frac{2}{x} + \frac{\sqrt{4 - x^2}}{x}} + C}\]

Do NOT forget to change the differential. Remember that in indefinite integrals, we drop absolute value bars for simplicity. Using your substitution, work backward to construct a right triangle. This procedure enables you to re-express trig functions as algebraic expressions in the final answer.

EXAMPLE 2
Using trigonometric substitution, show that \[\int \frac{1}{\sqrt{a^2 + x^2}} \, \dd x = \ln \abs{\sqrt{a^2 + x^2} + x} + C \comma\] where \(a \gt 0.\)

The expression \(\sqrt{a^2 + x^2}\) warrants the trig substitution \(x = a \tan \theta.\) Then \(\dd x = a \sec^2 \theta \, \dd \theta,\) and the integral becomes \[\int \frac{1}{\sqrt{a^2 + a^2 \tan^2 \theta}} \cdot a \sec^2 \theta \, \dd \theta = \int \frac{1}{\abs{\sec \theta}} \cdot \sec^2 \theta \, \dd \theta \period \] Since we're working with an indefinite integral, we drop the absolute value bars on the \(\abs{\sec \theta}\) for simplicity. We then get \[\int \sec \theta \, \dd \theta = \ln \abs{\sec \theta + \tan \theta} + C \period \]

Now we convert the answer in terms of \(x.\) Because \(x = a \tan \theta,\) it follows that \(\tan \theta = x/a.\) Thus, we construct a right triangle containing an angle \(\theta\) whose opposite side has length \(x\) and whose adjacent side has length \(a\) (Figure 2). We then see that \(\sec \theta = \sqrt{a^2 + x^2}/a\) and \(\tan \theta = x/a.\) Therefore, the integral becomes \[\ln \abs{\frac{\sqrt{a^2 + x^2}}{a} + \frac{x}{a}} + C_1 \period \] But we can re-express our answer as \[\ln \abs{\sqrt{a^2 + x^2} + x} - \ln a + C_1 \period\] Since \(a \gt 0\) is a constant, we can denote \(C_1 - \ln a\) as one constant: \(C.\) Our final answer is then \[\ln \abs{\sqrt{a^2 + x^2} + x} + C \comma \] as requested.

Changing Bounds

Let's now discuss how to change bounds when performing a trig substitution. This course of action requires us to solve for \(\theta,\) explaining the need for our substitutions to be invertible functions. Consider, as an example, the integral \[\int_{\sqrt 2}^{\sqrt 2/3} \sqrt{x^2 - 1} \, \dd x \period \] We perform the substitution \(x = \sec \theta,\) giving \(\dd x = \sec \theta \tan \theta \, \dd \theta.\) Now we change the bounds: For the lower bound, \(x = \sqrt 2,\) we find \[\sqrt 2 = \sec \theta \implies \theta = \frac{\pi}{4} \period\] Similarly, for the upper bound, \(x = \sqrt 2/3,\) we have \[\frac{\sqrt 2}{3} = \sec \theta \implies \theta = \frac{\pi}{3} \period\] Our new integral is then \[ \ba &\int_{\pi/4}^{\pi/3} \sqrt{\sec^2 \theta - 1} \sec \theta \tan \theta \, \dd \theta \\ = &\int_{\pi/4}^{\pi/3} \abs{\tan \theta} \sec \theta \tan \theta \, \dd \theta \period \ea \] How do we deal with \(\abs{\tan \theta} \ques\) Let's consider what values \(\abs{\tan \theta}\) outputs: For all \(\pi/4 \leq \theta \leq \pi/3,\) \(\tan \theta \gt 0.\) Since \(\tan \theta\) only takes on positive values, we can simply drop its absolute value bars. If our function only took on negative values, however, then we'd need to insert a negative sign after dropping the absolute value bars. Thus, the integral becomes \[\int_{\pi/4}^{\pi/3} \sec \theta \tan^2 \theta \, \dd \theta \period\] (We'll stop here, since this integral was for demonstration purposes.)

EXAMPLE 3
Evaluate \(\ds \int_{2/\sqrt 3}^{\sqrt 2} \frac{1}{x \sqrt{x^2 - 1}} \, \dd x\) using trigonometric substitution.

The root sign in \(\sqrt{x^2 - 1}\) can be stripped by letting \(x = \sec \theta.\) Then \(\dd x = \sec \theta \tan \theta \, \dd \theta.\) We now change the bounds: When \(x = 2/ \sqrt 3,\) we have \[\frac{2}{\sqrt 3} = \sec \theta \implies \theta = \frac{\pi}{6} \period\] Likewise, when \(x = \sqrt 2,\) we have \[2 = \sec \theta \implies \theta = \frac{\pi}{4} \period \] Therefore, our integral becomes \[ \ba &\int_{\pi/6}^{\pi/4} \frac{1}{\sec \theta \sqrt{\sec^2 \theta - 1}} \cdot \sec \theta \tan \theta \, \dd \theta \\ =&\int_{\pi/6}^{\pi/4} \frac{1}{\abs{\tan \theta}} \tan \theta \, \dd \theta \period \ea \] For convenience, let's drop the absolute value bars in \(\abs{\tan \theta}.\) The integral is then \[ \ba \int_{\pi/6}^{\pi/4} \dd \theta &= \theta \Big|_{\pi/6}^{\pi/4} \\[2ex] &= \frac{\pi}{4} - \frac{\pi}{6} \\[2ex] &= \boxed{\frac{\pi}{12}} \ea \]

Completing the Square

Consider the expression \(\sqrt{ax^2 + bx + c}.\) We complete the square to express \(ax^2 + bx + c\) in the form \((x - h)^2 + k,\) from which we substitute \(x - h = \sqrt k \tan \theta.\) For example, the quadratic \(x^2 - 4x + 10\) can be rewritten as \[x^2 - 4x + \underline{4} - \underline{4} + 10 = (x - 2)^2 + 6 \period\] Consequently, we can rewrite \[\int \sqrt{x^2 - 4x + 10} \, \dd x \qquad \text{as} \qquad \int \sqrt{(x - 2)^2 + 6} \, \dd x \comma\] and we can remove the root sign by letting \(x - 2 = \sqrt 6 \tan \theta.\) To find \(\dd x,\) we first write \(x\) explicitly: \[ \ba x &= 2 + \sqrt 6 \tan \theta \\ \dd x &= \sqrt 6 \sec^2 \theta \, \dd \theta \period \ea \] If the integral were definite, then we would change the bounds by isolating \(\theta:\) \[\frac{x - 2}{\sqrt 6} = \tan \theta \implies \theta = \tan^{-1} \left( \frac{x - 2}{\sqrt 6} \right) \period \] This procedure permits us to solve integrals that involve quadratics.

EXAMPLE 4
Evaluate \(\ds \int \frac{1}{\sqrt{x^2- 2x + 5}} \, \dd x\) using trigonometric substitution.

We complete the square under the square root to get \[\int \frac{1}{\sqrt{(x - 1)^2 + 4}} \, \dd x \period\] We let \(x - 1 = 2 \tan \theta.\) Then \(\dd x = 2 \sec^2 \theta \, \dd \theta.\) Afterward, the integral becomes \[ \ba &\int \frac{1}{\sqrt{4 \tan^2 \theta + 4}} \cdot 2 \sec^2 \theta \, \dd \theta \\ = &\int \frac{1}{2 \abs{\sec \theta}} \cdot 2 \sec^2 \theta \, \dd \theta \period \ea \] The integral is indefinite, so the absolute value bars around \(\abs{\sec \theta}\) can disappear. The integral is then \[\int \sec \theta \, \dd \theta = \ln \abs{\sec \theta + \tan \theta} + C \period\]

Now we express \(\sec \theta\) and \(\tan \theta\) in terms of \(x.\) When drawing the right triangle, note that \(\tan \theta = (x - 1)/2.\) Thus, our triangle should have legs of lengths \((x - 1)\) and \(2,\) which create a hypotenuse of length \(\sqrt{4 + (x - 1)^2}\) (Figure 3). Then \(\sec \theta = \sqrt{4 + (x - 1)^2}/2\) and \(\tan \theta = (x - 1)/2.\) Thus, our answer is \[ \ln \abs{\frac{\sqrt{4 + (x - 1)^2}}{2} + \frac{x - 1}{2}} + C_1 = \ln \abs{\sqrt{4 + (x - 1)^2} + x - 1} - \ln 2 + C_1 \period \] We can combine \(\ln 2 + C_1\) as simply one constant, \(C,\) so our final answer is \[\boxed{\ln \abs{\sqrt{4 + (x - 1)^2} + x - 1} + C}\]
