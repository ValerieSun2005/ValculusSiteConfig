Let \(f\) be the polynomial defined by \(f(x) = x^5 + 6x^3 + 4x + 2.\) Note that \(f(-1) = -9\) and \(f(0) = 2.\) Using Rolle's Theorem, show that \(f(x) = 0\) has exactly one real solution.
The function \(f(x)\) is a polynomial, so it is continuous for all \(x.\) Therefore, because \(f(-1) = -9\) and \(f(0) = 2,\) the Intermediate Value Theorem guarantees a solution to \(f(x) = 0\) within \(-1 \leq x \leq 2.\) We now show that \(f\) has no other zeros using a Proof By Contradiction: We first start by assuming that \(f\) has two zeros, \(x = a\) and \(x = b.\) It then follows that \[f(a) = f(b) = 0 \period\] Because \(f\) is a polynomial, it is continuous on \([a, b]\) and differentiable on \((a, b).\) Thus, Rolle's Theorem states that a value \(c\) in \((a, b)\) must satisfy \(f'(c) = 0.\) We then differentiate \(f(x)\) to find \[f'(c) = 5c^4 + 18c^2 + 4 = 0 \comma\] which has no solution. Therefore, we have a contradiction: \(f\) does not satisfy \(f(a) = f(b) = 0,\) so Rolle's Theorem cannot be used on \(f.\) We conclude, therefore, that \(f\) must only have one zero.
