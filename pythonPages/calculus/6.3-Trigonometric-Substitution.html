6.3 — Trigonometric Substitution

Introduction

We understand how to integrate \(\int x \sqrt{a^2 - x^2} \di x:\) simply substitute \(u = a^2 - x^2.\) But now consider the family of integrals \(\int \sqrt{a^2 - x^2} \di x,\) \(a \gt 0.\) To integrate this form, our plan is to remove this root sign. Let's substitute \(x = a \sin \theta.\) Then, using the identity \(1 - \sin^2 \theta = \cos^2 \theta,\) the integrand becomes \[\sqrt{a^2 - x^2} = \sqrt{a^2 - a^2 \sin \theta} = a \sqrt{1 - \sin^2 \theta} = a \abs{\cos \theta} \period\] This substitution eliminates the root sign, producing a simpler expression to integrate. We call this substitution procedure trigonometric substitution: the procedure by which we exploit trig identities to strip root signs.

Picking Substitutions

We've established that for the expression \(\sqrt{a^2 - x^2},\) which we'll denote Case 1, we should let \(x = a \sin \theta.\) But let's consider two more cases: Case 2, \(\sqrt{x^2 - a^2};\) and Case 3, \(\sqrt{a^2 + x^2}.\) For Case 2, we let \(x = a \sec \theta.\) Using the identity \(\sec^2 \theta - 1 = \tan^2 \theta,\) we have \[\sqrt{x^2 - a^2} = \sqrt{a^2 \sec^2 \theta - a^2} = a \sqrt{\sec^2 \theta - 1} = a \abs{\tan \theta} \period\] Similarly, for Case 3 we substitute \(x = a \tan \theta.\) We then use the identity \(\tan^2 \theta + 1 = \sec^2 \theta\) to get \[\sqrt{a^2 + x^2} = \sqrt{a^2 + a^2 \tan^2 \theta} = a \sqrt{1 + \tan^2 \theta} = a \abs{\sec \theta} \period\]

Note that in Case 1 we may also substitute \(x = a \cos \theta,\) which simplifies the radical to \(a \abs{\sin \theta}.\) This substitution accomplishes the same goal, eliminating the root. But when integrating, we compute the differential—in this case, \(\dd x = -a \sin \theta \di \theta,\) which has an extra negative sign. Similarly, using \(x = a \csc \theta\) for Case 2 and \(x = a \cot \theta\) for Case 3 both eliminate root signs, but their differentials contain negative signs. These substitutions are thereby more complicated, so we avoid them.

We must ensure that each substitution we make is one-to-one; that is, it has an inverse function. For example, when substituting \(x = a \sin \theta,\) \(x\) has an inverse function for \(- \pi/2 \leq \theta \leq \pi/2.\) Thus, we restrict \(\theta\) to the interval \([- \pi/2, \pi/2].\) Note that this interval matches the range of inverse sine. Conversely, in Case 2 we substitute \(x = a \sec \theta.\) Here, \(x\) is one-to-one for \(0 \leq \theta \lt \pi/2\) or \(\pi \leq \theta \leq 3 \pi/2,\) so we restrict \(\theta\) to either of these intervals. Finally, Case 3 presents the substitution \(x = a \tan \theta,\) which has an inverse function for \(- \pi/2 \lt \theta \lt \pi/2.\)

EXAMPLE 1
For each of (a)–(c), perform a trigonometric substitution to remove the root sign.
  1. \(\sqrt{25 - x^2}\)
  2. \(\sqrt{81 + x^2}\)
  3. \(\sqrt{x^2 - 100}\)
  1. This expression matches Case 1, so we substitute \(x = \sqrt{25} \sin \theta = 5 \sin \theta.\) We then get \[ \ba \sqrt{25 - (5 \sin \theta)^2} &= \sqrt{25 - 25 \sin^2 \theta} \\ &= 5 \sqrt{1 - \sin^2 \theta} \\ &= 5 \abs{\cos \theta} \period \ea \]
  2. Because this expression matches Case 3, we substitute \(x = \sqrt{81} \tan \theta = 9 \tan \theta.\) Then the expression becomes \[ \ba \sqrt{81 + (9 \tan \theta)^2} &= \sqrt{81 + 81 \tan^2 \theta} \\ &= 9 \sqrt{1 + \tan^2 \theta} \\ &= 9 \abs{\sec \theta} \period \ea \]
  3. We have a form of Case 3. Thus, we substitute \(x = \sqrt{100} \sec \theta = 10 \sec \theta,\) which yields \[ \ba \sqrt{(10 \sec \theta)^2 - 100} &= \sqrt{100 \sec^2 \theta - 100} \\ &= 10 \sqrt{\sec^2 \theta - 1} \\ &= 10 \abs{\tan \theta} \period \ea \]

Do NOT forget the absolute value bars. We'll discuss them later, when we integrate. But for now, as we work with indefinite integrals, we just drop the absolute value bars for simplicity. With definite integrals, however, we leave the absolute value bars and analyze the situation.

TRIGONOMETRIC SUBSTITUTIONS—INDEFINITE INTEGRALS
Suppose an indefinite integral contains a radical.
  1. Pick a trigonometric substitution to rewrite the radical. For example, the substitution \(x = 3 \sin \theta\) allows \(\sqrt{9 - x^2}\) to be rewritten as \(3 \abs{\sin \theta}.\) For simplicity, drop the absolute value bars.
  2. Compute the differential of the substitution. For example, the substitution \(x = 3 \sin \theta\) has the differential \(\dd x = 3 \cos \theta \di \theta.\)
  3. Re-express your answer in terms of the original variable. Draw a right triangle whose side lengths match the substitution you performed. For example, the reference triangle of the substitution \(x = 3 \sin \theta\) satisfies \(\sin \theta = x/3.\) Therefore, the triangle has a hypotenuse of length \(3\) and an angle \(\theta\) whose opposite side has length \(x.\)
EXAMPLE 2
Evaluate \(\ds \int \frac{1}{x \sqrt{4 - x^2}} \di x\) using a trigonometric substitution.

The denominator has the factor \(\sqrt{4 - x^2},\) whose root sign we clear by letting \(x = 2 \sin \theta:\) \[\sqrt{4 - (2 \sin \theta)^2} = \sqrt{4 - 4 \sin^2 \theta} = 2 \sqrt{1 - \sin^2 \theta} = 2 \abs{\cos \theta} \period \] Additionally, \(\dd x = 2 \cos \theta \di \theta.\) The integral then becomes \[ \ba &\int \frac{1}{(2 \sin \theta) \sqrt{4 - 4 \sin^2 \theta}} \cdot 2 \cos \theta \di \theta \\ = &\int \frac{2 \cos \theta}{(2 \sin \theta)(2 \abs{\cos \theta})} \di \theta \period \ea \] Notice that the new integrand has no root signs, so it's a less complicated expression. Since we have an indefinite integral, we ignore the absolute value bars of \(\abs{\cos \theta}.\) This assumption is merely for simplicity. The integral is then \[\int \frac{1}{2} \csc \theta \di \theta \period\] Recall that \(\int \csc \theta \di \theta = - \ln \abs{\csc \theta + \cot \theta} + C.\) Thus, we have \[\int \frac{1}{2} \csc \theta \di \theta = -\frac{1}{2} \ln \abs{\csc \theta + \cot \theta} + C \period\]

At this stage, we convert our answer back to \(x \col\) Since \(x = 2 \sin \theta,\) it follows that \(\sin \theta = x/2.\) We then reference a right triangle that satisfies this description: The triangle has a hypotenuse of length \(2\) and an angle \(\theta\) whose opposite side is \(x\) (Figure 1). These lengths result in the side adjacent to \(\theta\) having length \(\sqrt{2^2 - x^2} = \sqrt{4 - x^2}.\) We then see that \(\csc \theta = 2/x\) and \(\cot \theta = \sqrt{4 - x^2}/x.\) Therefore, our final answer is \[\boxed{-\frac{1}{2} \ln \abs{\frac{2}{x} + \frac{\sqrt{4 - x^2}}{x}} + C}\]

Do NOT forget to change the differential. Do not blindly switch from \(\dd x\) to \(\dd \theta.\) Moreover, do not leave your answer in terms of \(\theta;\) instead, use your substitution to to construct a reference triangle. This procedure enables you to re-express trig functions as algebraic expressions in the final answer.

EXAMPLE 3
Using a trigonometric substitution, show that \[\int \frac{1}{\sqrt{a^2 + x^2}} \di x = \ln \abs{\sqrt{a^2 + x^2} + x} + C \comma\] where \(a \gt 0.\)

The expression \(\sqrt{a^2 + x^2}\) warrants the trig substitution \(x = a \tan \theta:\) \[\sqrt{a^2 + (a \tan \theta)^2} = \sqrt{a^2 + a^2 \tan^2 \theta} = a \sqrt{1 + \tan^2 \theta} = a \abs{\sec \theta} \period \] Then \(\dd x = a \sec^2 \theta \di \theta,\) and the integral becomes \[\int \frac{1}{\sqrt{a^2 + a^2 \tan^2 \theta}} \cdot a \sec^2 \theta \di \theta = \int \frac{1}{\abs{\sec \theta}} \cdot \sec^2 \theta \di \theta \period \] Since we're working with an indefinite integral, we drop the absolute value bars on the \(\abs{\sec \theta}\) for simplicity. We then get \[\int \sec \theta \di \theta = \ln \abs{\sec \theta + \tan \theta} + C \period \]

Now we convert the answer in terms of \(x.\) Because \(x = a \tan \theta,\) it follows that \(\tan \theta = x/a.\) Thus, we construct a right triangle containing an angle \(\theta\) whose opposite side has length \(x\) and whose adjacent side has length \(a\) (Figure 2). We then see that \(\sec \theta = \sqrt{a^2 + x^2}/a\) and \(\tan \theta = x/a.\) Therefore, the integral becomes \[\ln \abs{\frac{\sqrt{a^2 + x^2}}{a} + \frac{x}{a}} + C_1 \period \] But we can re-express our answer as \[\ln \abs{\sqrt{a^2 + x^2} + x} - \ln a + C_1 \period\] Since \(a \gt 0\) is a constant, we denote \(C_1 - \ln a\) as one constant: \(C.\) Our final answer is then \[\ln \abs{\sqrt{a^2 + x^2} + x} + C \comma \] as requested.

Changing Bounds

Let's now discuss how to change bounds when performing a trig substitution. For example, consider the integral \[\int_{1/2}^{\sqrt 3/2} \sqrt{1 - x^2} \di x \period \] We perform the substitution \(x = \sin \theta,\) giving \(\dd x = \cos \theta \di \theta.\) Now we change the bounds: For the lower bound, \(x = 1/2,\) we find \[1/2 = \sin \theta \implies \theta = \pi/6 \period \] Likewise, for the upper bound, \(x = \sqrt 3/2,\) we get \[ \sqrt 3/2 = \sin \theta \implies \theta = \pi/3 \period \] In changing the bounds, we solve for \(\theta,\) which explains why our substitutions need to have inverse functions. Thus, our new integral is \[ \ba &\int_{\pi/6}^{\pi/3} \sqrt{1 - \sin^2 \theta} \, \cos \theta \di \theta \period \\ = &\int_{\pi/6}^{\pi/3} \abs{\cos \theta} \, \cos \theta \di \theta \period \ea \] How do we deal with \(\abs{\cos \theta} \ques\) Let's consider the values \(\cos \theta\) outputs: For all \(\pi/6 \lt \theta \lt \pi/3,\) \(\cos \theta \gt 0.\) Since \(\cos \theta\) only takes on positive values on this interval, it follows that \(\abs{\cos \theta} = \cos \theta.\) Therefore, we replace \(\abs{\cos \theta}\) with \(\cos \theta\) to attain \[ \ba \int_{\pi/6}^{\pi/3} \cos^2 \theta \di \theta &= \int_{\pi/6}^{\pi/3} \frac{1}{2} \par{1 + \cos 2 \theta} \di \theta \\ &= \par{\frac{1}{2} \theta + \frac{1}{4} \sin 2 \theta} \Big|_{\pi/6}^{\pi/3} \\ &= \par{\frac{1}{2} \cdot \frac{\pi}{3} + \frac{1}{4} \sin \frac{2 \pi}{3}} - \par{\frac{1}{2} \cdot \frac{\pi}{6} + \frac{1}{4} \sin \frac{\pi}{3} } \\ &= \boxed{\frac{\pi}{12}} \ea \] After changing the integral's bounds, we did not return to \(x.\) We performed all remaining calculations in terms of \(\theta.\)

TRIGONOMETRIC SUBSTITUTIONS—DEFINITE INTEGRALS
Suppose a definite integral contains a radical.
  1. Pick a trigonometric substitution to rewrite the radical.
  2. Compute the differential of the substitution.
  3. Change the bounds on the integral. Perform all remaining calculations in terms of \(\theta.\) Do not convert back to the original variable.
EXAMPLE 4
Evaluate \(\ds \int_{-\sqrt 2}^{-2/\sqrt 3} \frac{1}{x \sqrt{x^2 - 1}} \di x\) using a trigonometric substitution.

The root sign in \(\sqrt{x^2 - 1}\) is stripped by letting \(x = \sec \theta:\) \[\sqrt{\sec^2 \theta - 1} = \sqrt{\tan^2 \theta} = \abs{\tan \theta} \period\] Then \(\dd x = \sec \theta \tan \theta \di \theta.\) We now change the bounds: When \(x = -\sqrt 2,\) we have \[- \sqrt 2 = \sec \theta \implies \theta = \frac{3 \pi}{4} \period \] Likewise, when \(x = -2/ \sqrt 3,\) we have \[-\frac{2}{\sqrt 3} = \sec \theta \implies \theta = \frac{5 \pi}{6} \period\] Therefore, our integral becomes \[ \ba &\int_{3 \pi/4}^{5 \pi/6} \frac{1}{\sec \theta \sqrt{\sec^2 \theta - 1}} \cdot \sec \theta \tan \theta \di \theta \\ =&\int_{3 \pi/4}^{5 \pi/6} \frac{1}{\abs{\tan \theta}} \, \tan \theta \di \theta \period \ea \] How do we deal with \(\abs{\tan \theta} \ques\) We consider the values \(\tan \theta\) outputs for \(3 \pi/4 \lt \theta \lt 5 \pi/6:\) On this interval, \(\tan \theta\) is strictly negative. It then follows that \(\abs{\tan \theta} = - \tan \theta,\) so we replace \(\abs{\tan \theta}\) with \((- \tan \theta)\) to get \[ \ba \int_{3 \pi/4}^{5 \pi/6} - \dd \theta &= - \theta \Big|_{3 \pi/4}^{5 \pi/6} \\[2ex] &= - \par{\frac{5 \pi}{6} - \frac{3 \pi}{4}} \\[2ex] &= \boxed{-\frac{\pi}{12}} \ea \] Since we already changed the bounds on the integral, we did NOT re-express our antiderivative in terms of \(x.\) Therefore, we don't need to draw a triangle, the procedure we use in indefinite integrals to express our final answer in terms of \(x.\)

Coefficients in \(x^2\)

Consider, as an example, the expression \(\sqrt{a^2 + b^2 x^2},\) where \(a\) and \(b\) are both positive constants. To remove the root sign, we substitute \(x = \frac{a}{b} \tan \theta:\) \[\sqrt{a^2 + b^2 \par{\frac{a}{b} \tan \theta}^2} = \sqrt{a^2 + a^2 \tan^2 \theta} = a \abs{\sec \theta} \period\]

EXAMPLE 5
Using a trigonometric substitution, evaluate \(\ds \int \frac{x^3}{(9x^2 + 25)^{3/2}} \di x.\)

We substitute \(x = \frac{5}{3} \tan \theta\) to rewrite the denominator of the integrand as \[\sqrt{9 \par{\frac{5}{3} \tan \theta}^2 + 25 } = \sqrt{25 \tan^2 \theta + 25} = 5 \abs{\sec \theta} \period\] Then \(\dd x = \frac{5}{3} \sec^2 \theta \di \theta.\) The integral becomes \[\int \frac{\par{\frac{5}{3} \tan \theta}^3}{27 \cdot \par{\frac{5}{3} \abs{\sec \theta}}^3 } \cdot \frac{5}{3} \sec^2 \theta \di \theta \period\] Because we have an indefinite integral, we drop the absolute value bars in \(\abs{\sec \theta}:\) \[\frac{5}{81} \int \frac{\tan^3 \theta}{\sec^3 \theta} \, \sec^2 \theta \di \theta \period\] In the integrand, there lies the extra factor \(\sec \theta \tan \theta:\) \[\frac{5}{81} \int \frac{\tan \theta \tan^2 \theta}{\sec^2 \theta} \, \sec \theta \di \theta = \frac{5}{81} \int \frac{\tan \theta (\sec^2 \theta - 1)}{\sec^2 \theta} \, \sec \theta \di \theta \period\] Therefore, we substitute \(u = \sec \theta\) because then \(\dd u = \sec \theta \tan \theta \di \theta.\) The substitution then yields \[ \ba \frac{5}{81} \int \frac{u^2 - 1}{u^2} \di u &= \frac{5}{81} \int 1 - \frac{1}{u^2} \di u \\ &= \frac{5}{81} \par{u + \frac{1}{u}} + C \\ &= \frac{5}{81} \par{\sec \theta + \frac{1}{\sec \theta}} + C \period \ea \]

Finally, we convert back to \(x.\) Since \(x = \frac{5}{3} \tan \theta,\) it follows that \(\tan \theta = 3x/5.\) Therefore, we construct a right triangle that has an angle \(\theta\) whose opposite side has length \(3x\) and whose adjacent side has length \(5.\) Then the hypotenuse has length \(\sqrt{(3x)^2 + 5^2} = \sqrt{9x^2 + 25}\) (Figure 3). Thus, \(\sec \theta = \sqrt{9x^2 + 25}/5.\) Our final answer is \[\boxed{\frac{5}{81} \par{\frac{\sqrt{9x^2 + 25}}{5} + \frac{5}{\sqrt{9x^2 + 25}}} + C }\]

Completing the Square

Consider the expression \(\sqrt{x^2 + bx + c}.\) We complete the square to express \(x^2 + bx + c\) in the form \((x - h)^2 + k,\) from which we substitute \(x - h = \sqrt k \tan \theta.\) For example, the quadratic \(x^2 - 4x + 10\) can be rewritten as \[x^2 - 4x + \underline{4} - \underline{4} + 10 = (x - 2)^2 + 6 \period\] Consequently, we can rewrite \[\int \sqrt{x^2 - 4x + 10} \di x \qquad \text{as} \qquad \int \sqrt{(x - 2)^2 + 6} \di x \comma\] and we can remove the root sign by letting \(x - 2 = \sqrt 6 \tan \theta.\) To find \(\dd x,\) we first write \(x\) explicitly: \[ \ba x &= 2 + \sqrt 6 \tan \theta \\ \dd x &= \sqrt 6 \sec^2 \theta \di \theta \period \ea \] If the integral were definite, then we would change the bounds by isolating \(\theta:\) \[\frac{x - 2}{\sqrt 6} = \tan \theta \implies \theta = \tan^{-1} \left( \frac{x - 2}{\sqrt 6} \right) \period \] This procedure permits us to solve integrals that involve quadratics.

EXAMPLE 6
Evaluate \(\ds \int \frac{1}{\sqrt{x^2- 2x + 5}} \di x\) using a trigonometric substitution.

We complete the square under the square root to get \[\int \frac{1}{\sqrt{(x - 1)^2 + 4}} \di x \period\] We let \(x - 1 = 2 \tan \theta.\) Then \[ \ba x &= 1 + 2 \tan \theta \\ \dd x &= 2 \sec^2 \theta \di \theta \period \ea \] Afterward, the integral becomes \[ \ba &\int \frac{1}{\sqrt{4 \tan^2 \theta + 4}} \cdot 2 \sec^2 \theta \di \theta \\ = &\int \frac{1}{2 \abs{\sec \theta}} \cdot 2 \sec^2 \theta \di \theta \period \ea \] The integral is indefinite, so we drop the absolute value bars around \(\abs{\sec \theta}.\) The integral is then \[\int \sec \theta \di \theta = \ln \abs{\sec \theta + \tan \theta} + C \period\]

Now we express \(\sec \theta\) and \(\tan \theta\) in terms of \(x.\) When drawing the right triangle, note that \(\tan \theta = (x - 1)/2.\) Thus, our triangle should have legs of lengths \((x - 1)\) and \(2,\) which create a hypotenuse of length \(\sqrt{4 + (x - 1)^2}\) (Figure 4). Then \(\sec \theta = \sqrt{4 + (x - 1)^2}/2,\) and we've established that \(\tan \theta = (x - 1)/2.\) Thus, our answer is \[ \ln \abs{\frac{\sqrt{4 + (x - 1)^2}}{2} + \frac{x - 1}{2}} + C_1 = \ln \abs{\sqrt{4 + (x - 1)^2} + x - 1} - \ln 2 + C_1 \period \] We combine \(\ln 2 + C_1\) as simply one constant, \(C,\) so our final answer is \[\boxed{\ln \abs{\sqrt{4 + (x - 1)^2} + x - 1} + C}\]

SECTION SUMMARY
  • Consider the following forms and substitution choices.
    • For \(\sqrt{a^2 - x^2},\) substitute \(x = a \sin \theta. \)
    • For \(\sqrt{x^2 - a^2},\) substitute \(x = a \sec \theta.\)
    • For \(\sqrt{a^2 + x^2},\) substitute \(x = a \tan \theta.\)
  • In an indefinite integral, drop absolute value bars. For this cases, draw a right triangle whose side lengths match the substitution you made.
  • In a definite integral, change the bounds of integration by solving for \(\theta\) in the substitution you performed. Be careful when dropping absolute value bars: consider whether the function in absolute value bars is strictly positive or strictly negative on the interval through which you're integrating (see Example 4).
  • The family of functions \(\sqrt{x^2 + bx + c}\) can be rewritten as \(\sqrt{(x - h)^2 + k}\) by completing the square. Then substitute \(x - h = \sqrt k \tan \theta\) (see Example 6).
