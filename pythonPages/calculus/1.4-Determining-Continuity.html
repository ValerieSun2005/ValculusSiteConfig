1.4 — Determining Continuity

In this section, we formally define continuity using limits and connect continuity to families of functions. We classify types of discontinuities.

Definition of Continuity

A function \(f\) is continuous at a value \(a\) if the graph of \(y = f(x)\) is smooth at \(x = a.\) In other words, if we can trace the curve of \(y = f(x)\) at \(x = a\) without lifting our pen, then \(f(x)\) is continuous at \(x = a.\) Formally, we say \(f(x)\) is continuous at \(x = a\) if \[\lim_{x \to a} f(x) = f(a) \period \]

DEFINITION OF CONTINUITY

The function \(f(x)\) is continuous at \(x = a\) if \[\lim_{x \to a} f(x) = f(a) \period \]

STEPS FOR DETERMINING CONTINUITY
To determine whether \(f(x)\) is continuous at \(x = a\):
  1. Establish that \(f(x)\) is defined at \(x = a.\)
  2. Determine whether \(\lim_{x \to a} f(x)\) exists.
  3. Verify that \(\lim_{x \to a} f(x) = f(a).\)
At an endpoint, it is sufficient to only consider the appropriate one-sided limit: If \(a\) is a left endpoint, then \(f(x)\) is continuous at \(x = a\) if \(\lim_{x \to a^+} f(x) = f(a).\) If \(a\) is a right endpoint, then \(f(x)\) is continuous at \(x = a\) if \(\lim_{x \to a^-} f(x) = f(a).\)
CONTINUITY OVER AN INTERVAL

If a function \(f(x)\) is continuous at every point in \(a \leq x \leq b,\) then \(f(x)\) is continuous over \(a \leq x \leq b.\) At the endpoints—\(x = a\) and \(x = b\)— \(f(x)\) is continuous from the right and left, respectively.

Classifying Discontinuities

We now learn to classify discontinuities. We consider three types of discontinuities: removable discontinuities, jump discontinuities, and infinite discontinuities.

A function \(f(x)\) has a removable discontinuity—also called a hole—at at \(x = a\) if \(\lim_{x \to a} f(x)\) exists but does not equal \(f(a).\) Alternatively, \(f(a)\) may be undefined. In either case, the graph of \(y = f(x)\) at \(x = a\) has a hole—a single point at which \(f(x)\) is discontinuous. In Figure 1, the graph of \(y = f(x)\) has a removable discontinuity at \(x = a\) because \[\lim_{x \to a} f(x) = K \ne f(a) \period \] We call this discontinuity removable because we may fill in the hole at \((a, K)\) to restore continuity. It's also possible to define a new function \(g\) that matches \(f\) but is continuous at \(a;\) namely, we let \[g(x) = \begin{cases} f(x) &x \ne a \nl K &x = a \period \end{cases} \]

The function \(f(x)\) has a jump discontinuity at \(x = a\) if \(\lim_{x \to a} f(x)\) does not exist but \(f(a)\) exists and both \[\lim_{x \to a^-} f(x) \and \lim_{x \to a^+} f(x)\] exist and are finite (Figure 2).

The name is self-explanatory: a jump breaks the continuity in the graph of \(y = f(x)\) when \(x = a.\)

Lastly, the function \(f(x)\) has an infinite discontinuity at \(x = a\) if either \[\lim_{x \to a^-} f(x) \or \lim_{x \to a^+} f(x)\] is infinite. At \(x = a,\) the graph of \(y = f(x)\) has a vertical asymptote: \(f\) is unbounded on either side of this line (Figure 3).

TYPES OF DISCONTINUITIES

  1. \(f(x)\) has a removable discontinuity (or a hole) at \(x = a\) if \(\lim_{x \to a} f(x)\) exists but does not equal \(f(a).\)
  2. \(f(x)\) has a jump discontinuity at \(x = a\) if \(\lim_{x \to a} f(x)\) does not exist.
  3. \(f(x)\) has an infinite discontinuity at \(x = a\) if either \[\lim_{x \to a^-} f(x) \or \lim_{x \to a^+} f(x)\] is infinite.

EXAMPLE 1
For each of (i)–(iii), classify the discontinuity of \(f(x)\) at the given value of \(x.\)
  1. \(\ds f(x) = \frac{x^2 - 16}{x + 4}\) at \(x = -4\)
  2. \(\ds f(x) = \begin{cases} x - 2 &x \lt 2 \nl 3(3 - x)^2 &x \geq 2 \end{cases} \) at \(x = 2\)
  3. \(\ds f(x) = \frac{1}{x + 3}\) at \(x = -3\)
  1. The numerator of \(f(x)\) is a difference of squares: \[x^2 - 16 = (x + 4)(x - 4) \period \] Thus, we rewrite \(f(x)\) as \[\frac{(x + 4)(x - 4)}{x + 4)} = x - 4 \comma x \ne -4 \period\] We then see \[\lim_{x \to -4} f(x) = -8 \period\] Since \(f(-4)\) is undefined, \(\lim_{x \to -4} f(x) \ne f(-4) \) and so \(f(x)\) has a removable discontinuity at \(x = -4.\)
  2. We find \[ \ba \lim_{x \to 2^-} f(x) &= 2 - 2 \nl &= 0 \comma \ea \]\ and \[ \ba \lim_{x \to 2^+} f(x) &= 3(3 - 2)^2 \nl &= 3 \period \ea \] Since these one-handed limits are not equivalent, \(\lim_{x \to 2} f(x)\) does not exist. Therefore, this discontinuity is a jump discontinuity.
  3. The graph of \(y = 1/(x + 3)\) has a vertical asymptote at \(x = -3.\) Therefore, \(x = -3\) is an infinite discontinuity of \(f(x).\)

Continuity of Piecewise Functions

We now provide an addition to Part (ii) of Example 1. A piecewise function \[ f(x) = \bc g(x) &x \leq a \nl h(x) &x \gt a \ec \or f(x) = \bc g(x) &x \lt a \nl h(x) &x \geq a \ec \] is continuous at \(x = a\) if \[\lim_{x \to a} f(x) = f(a) \period\] The limit \(\lim_{x \to a} f(x)\) exists if the left- and right-hand limits are equivalent—that is, if \[\lim_{x \to a^-} g(x) = \lim_{x \to a^+} h(x) \period\]

EXAMPLE 2
Let \[ f(x)= \bc kx + 4 &x \leq 3 \nl 2k - 5 &x \gt 3 \period \ec \] Find \(k\) such that \(f(x)\) is continuous at \(x = 3.\)

We see \[f(3) = 3k + 4 \comma \] the value \(\lim_{x \to 3} f(x)\) must equal if \(f(x)\) is continuous at \(x = 3.\) The limit \(\lim_{x \to 3} f(x)\) exists if its one-sided limits are equivalent—namely, if \[\lim_{x \to 3^-} f(x) = \lim_{x \to 3^+} f(x) \period \] \[ 3k + 4 = 2k - 5 \implies k = -9 \period\]

Now we check: For \(k = -9,\) \[f(3) = 3(-9) + 4 = -23 \period \] Also, \(\lim_{x \to 3} f(x) = -23,\) matching the value of \(f(3).\) Thus, for \(\boxed{k = -9,}\) \(f(x)\) is continuous at \(x = 3.\)

EXAMPLE 3
Let \[ g(t)= \bc t^2- 4 &t \lt a \nl -2 - t &t \geq a \period \ec \] Find \(a\) such that \(g(t)\) is continuous at \(t = a.\)
The function \(g(t)\) is continuous at \(t = a\) if \[\lim_{t \to a^-} g(t) = \lim_{t \to a^+} g(t) \period\] The left-hand expression is \((t^2 - 4,)\) and the right-hand expression is \((-2 - t).\) Thus, we find \[ \ba a^2 - 4 &= -2 - a \nl a^2 + a - 2 &= 0 \nl (a - 2)(a + 1) &= 0 \comma \ea \] whose solutions are \[\boxed{a = -1 \and a = 2}\]

Continuity of Combinations of Functions

We now discuss the continuity of combinations of functions. We begin by stating that if \(f\) and \(g\) are continuous at \(a,\) then \((f + g),\) \((f - g),\) \((f \cdot g),\) and [if \(g(a) \ne 0\)] \(f/g\) are all continuous at \(a.\) The proof of this fact is found here.

THEOREM 1: CONTINUITY OF COMBINATIONS OF FUNCTIONS
If \(f(x)\) and \(g(x)\) are both continuous at \(x = a,\) then the following combinations of \(f(x)\) and \(g(x)\) are also continuous at \(x = a.\)
  1. \(f(x) + g(x)\)
  2. \(f(x) - g(x)\)
  3. \(f(x) \cdot g(x)\)
  4. \(f(x)/g(x) \;\) if \(\; g(a) \ne 0.\)

We now determine the continuity of composite functions. If \(f\) and \(g\) are functions, then composite functions of \(f\) and \(g\) are \[h(x) = f(g(x)) \and h(x) = g(f(x)) \period\] Let us denote \(h\) as either form. If \(h(x)\) is continuous at \(x = a,\) then we must have \begin{align} \lim_{x \to a} h(x) &= h(a) \nonumber \nl \lim_{x \to a} f(g(x)) &= f(g(a)) \label{eq:comp-funct-continuous} \period \end{align} The conditions to satisfy \(\eqRefer{eq:comp-funct-continuous}\) are as follows: \(g(x)\) is continuous at \(x = a\) and \(f(x)\) is continuous at \(x = g(a).\) Then \[\lim_{x \to a} f(g(x)) = f \left(\lim_{x \to a} g(x)\right) = f(g(a)) \comma \] thus showing \(h(x)\) is continuous at \(x = a.\)

THEOREM 2: CONTINUITY OF COMPOSITE FUNCTIONS
If \(g(x)\) is continuous at \(x = a\) and \(f(x)\) is continuous at \(x = g(a),\) then \(f(g(x))\) is continuous at \(x = a.\)

Continuity of Certain Functions

Many functions we know are continuous on their domain. For example, polynomials are defined and continuous for all real numbers. Conversely, logarithms are defined and continuous for strictly positive inputs. Table 1 provides the intervals of continuity for many functions we've seen: polynomial, rational, radical, absolute value, exponential, logarithmic, trigonometric, and inverse trigonometric functions.

Consider the example \[h(x) = \ln(x - 2) \comma \] which is in the form \(f(g(x))\) where \[f(x) = \ln x \and g(x) = x - 2 \period\] Since logarithms are only defined for positive inputs, \(\ln x\) is continuous for \(x \gt 0.\) Therefore, the nested function \(g(x)\) must follow this restriction and be greater than \(0.\) We find \[ \begin{aligned} g(x) &\gt 0 \\ x - 2 &\gt 0 \\ x &\gt 2 \period \end{aligned} \] The function \(g(x) = x - 2\) is continuous for all \(x,\) so \(\ln(x - 2)\) is continuous for all \(x \gt 2\) (Theorem 2).

EXAMPLE 4
For each of (i)–(iii), determine the interval on which \(f(x)\) is continuous.
  1. \(\ds f(x) = \frac{\cos x + x}{\ln x}\)
  2. \(\ds f(x) = \frac{\tan x}{x + 2}\)
  3. \(\ds f(x) = e^{\sqrt{3 - x}}\)
  1. The numerator is a sum of two continuous functions—\(\cos x\) and \(x\)—and is therefore continuous for all \(x.\) Conversely, the denominator \(\ln x\) is only continuous for \(x \gt 0.\) Therefore, for \(x \gt 0\) the numerator and denominator are both continuous and so \(f(x)\) is continuous for \(x \gt 0.\)
  2. We find the values of \(x\) such that the numerator and denominator of \(f(x)\) are both defined (and the denominator \(\ne 0\)). On this same interval we establish continuity for \(f(x)\) since \(\tan x\) and \((x + 2)\) are continuous for all \(x\) in their domain. The function \(\tan x\) is continuous for all \[ x \ne \pm \frac{\pi}{2} \comma \pm 3 \frac{\pi}{2} \comma \pm 5 \frac{\pi}{2} \comma \dots \period \] Also, \(x - 2 = 0\) at \(x = 2.\) Therefore, \(f(x)\) is discontinuous at \[x = -2 \and x = \frac{\pi}{2} \pm \pi n \comma n = 0, \pm 1, \pm 2, \dots \period \]
  3. The function \(e^x\) is continuous for all \(x.\) But we have \(e^{\sqrt{3 - x}},\) and \(\sqrt{3 - x}\) is continuous for \(x \leq 3.\) Thus, \(f(x)\) is continuous for all \(x \leq 3.\)
